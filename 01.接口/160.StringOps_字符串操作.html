StringOps_字符串操作 ​

模块简介 ​

StringOps 是字符串操作工具模块，封装了常用的字符串处理功能，包括字符串验证、转换、格式化、JSON 处理、相似度计算等。

功能概览 ​

  • 字符串验证：判断是否为空/空白、是否包含中文、是否为纯数字、提取数字、身份证成年校验、是否为普通字面值字符串
  • 字符串比较：区分大小写/忽略大小写的相等判断（equals、equalsIgnoreCase）
  • 字符串转换：首字母大小写转换、反转、全角半角互转、汉字转拼音（全拼/首字母）、null 转空串
  • 字符串处理：获取长度、去除空格与标点符号、按索引取分割项
  • URL 提取：从文本中提取所有网址（JSON 数组）
  • JSON 处理：字符串拆分、JSON 数组/对象内字符串数字转整数、去重、随机选择、判断是否包含
  • 数组操作：原地反转字符串数组（reverseStringArrayInPlace）
  • 相似度计算：Jaro-Winkler 相似度算法、汉明距离

8. 相似度计算 ​

jaroWinklerSim() ​

计算两个字符串的 Jaro-Winkler 相似度。

推荐用于等长字符串的比较。

Jaro-Winkler 相似度是 Jaro 相似度的改进版本，对前缀匹配给予更高权重。返回值范围：0.0（完全不同）到 1.0（完全相同）。

参数名类型必填说明
s1String?是第一个字符串
s2String?是第二个字符串
返回值类型说明
Double相似度值（0.0 ~ 1.0），若任一输入为 null 则返回 0.0

示例：

lua
local strOps = UtilsMain.stringOps()
local similarity1 = strOps.jaroWinklerSim("hello", "hello")
print(similarity1)  -- 输出: 1.0

local similarity2 = strOps.jaroWinklerSim("hello", "hallo")
print(similarity2)  -- 输出: 约 0.88

local similarity3 = strOps.jaroWinklerSim("hello", "world")
print(similarity3)  -- 输出: 约 0.47

hmDistance() ​

计算两个等长字符串的汉明距离（不同字符位置的个数）。

汉明距离表示两个等长字符串对应位置上不同字符的数量。例如 "110011" 与 "100010" 在第3位和第5位不同，汉明距离为 2。

参数名类型必填说明
s1String?是第一个字符串
s2String?是第二个字符串
返回值类型说明
Int汉明距离（不同字符位置的个数）；若任一输入为 null、或长度不相等，返回 -1

示例：

lua
local strOps = UtilsMain.stringOps()

-- 等长字符串比较
local dist1 = strOps.hmDistance("110011", "100010")
print(dist1)  -- 输出: 2（第3位和第5位不同）

-- 完全相同
local dist2 = strOps.hmDistance("hello", "hello")
print(dist2)  -- 输出: 0

-- 长度不等
local dist3 = strOps.hmDistance("abc", "abcd")
print(dist3)  -- 输出: -1

-- null 输入
local dist4 = strOps.hmDistance("abc", nil)
print(dist4)  -- 输出: -1

// 版本：Pro 9.3.11-0
toastLog("Hello, xfxPlugin");

var UtilsMain = {
  dcl: null,
  UtilsMainClass: null, // 缓存类对象

  initialization: function (dexPath, soBaseDir, ctx) {
    // 1. 路径校验
    if (!files.exists(dexPath)) throw new Error("Dex不存在：" + dexPath);
    console.log("Dex存在：" + dexPath);

    var cpuAbi = android.os.Build.CPU_ABI;
    console.log("CPU ABI：" + cpuAbi);
    var soDir = soBaseDir + cpuAbi + "/";
    if (!files.exists(soDir)) throw new Error("SO目录不存在：" + soDir);
    console.log("SO目录存在：" + soDir);

    // 2. 构建类加载器
    this.dcl = this._getDexClassLoader(dexPath, soDir);
    console.log("类加载器构建成功");

    // 3. 获取插件入口类
    try {
      this.UtilsMainClass = this.dcl.loadClass("com.xfx.plugin.UtilsMain");
      toastLog("成功加载UtilsMain类");
    } catch (e) {
      throw new Error("加载UtilsMain失败：" + e.message);
    }
    

    // 4. 初始化插件
    try {
      // 尝试静态init
      var contextClass = java.lang.Class.forName("android.content.Context");
      var initMethod = this.UtilsMainClass.getMethod("init", contextClass);
      initMethod.invoke(null, ctx);
      toastLog("xfx插件初始化成功（静态方法）");
    } catch (staticErr) {
      toastLog("xfx插件初始化失败：" + staticErr.message);
    }
  },
  _getDexClassLoader: function (dexPath, soDir) {
    if (this.dcl) return this.dcl;

    var dexOptDir = context
      .getDir("dex", android.app.Activity.MODE_PRIVATE)
      .getAbsolutePath();
    var jniPrivateDir = context.getDir(
      "libs",
      android.app.Activity.MODE_PRIVATE
    );
    files.removeDir(jniPrivateDir);
    var newJniDir = jniPrivateDir + "/" + new Date().getTime() + "/";
    files.createWithDirs(newJniDir);

    // 拷贝SO
    var soList = files.listDir(soDir, function (f) {
      return f.endsWith(".so");
    });
    // if (soList.length === 0) throw new Error("SO目录无文件：" + soDir);
    for (var i = 0; i < soList.length; i++) {
      var soSrc = new java.io.File(soDir, soList[i]).getAbsolutePath();
      var soDest = new java.io.File(newJniDir, soList[i]).getAbsolutePath();
      files.copy(soSrc, soDest);
      if (!files.exists(soDest)) throw new Error("SO拷贝失败：" + soSrc);
    }

    // 构建类加载器
    var parentCL = context.getClass().getClassLoader();
    var dcl = new Packages.dalvik.system.DexClassLoader(
      dexPath,
      dexOptDir,
      newJniDir,
      parentCL
    );
    this.dcl = dcl;
    return dcl;
  },

  /**
   * 智能获取StringOps实例（适配静态/实例方法）
   */
  getStringOps: function () {
    if (!this.UtilsMainClass) throw new Error("请先初始化插件");

    // 检查是否有静态stringOps方法
    try {
      var stringOpsMethod = this.UtilsMainClass.getMethod("stringOps");
      var stringOpsObj = stringOpsMethod.invoke(null);
      if (stringOpsObj) {
        toastLog("通过静态方法获取StringOps成功");
        return stringOpsObj;
      }
    } catch (staticErr) {
        throw new Error("获取StringOps失败：静态方法不存在！错误详情：" + staticErr.message);
    }
  },
  destroy: function () {
    this.dcl = null;
    this.UtilsMainClass = null;
  },
};

// ====================== 调用示例 ======================

示例 4：模糊搜索匹配 ​

lua
local xfxModule = require('lib/XfxPlugin')
local XFX = xfxModule:new({apkName = 'xfxPlugin-release.apk'})

local strOps = XFX:getOps('stringOps')

if not strOps then
    print('无法获取 strOps 对象')
    return
end

-- 搜索关键词
local keyword = 'hello world'

-- 候选列表
local candidates = {
    'hello world',
    'hello word',
    'hallo world',
    'world hello',
    'hello',
    'world',
    'goodbye world'
}

-- 设置相似度阈值
local threshold = 0.7

print('搜索关键词：' .. keyword)
print('相似度阈值：' .. threshold)
print('\n匹配结果：')

-- 计算相似度并筛选
local results = {}
for i = 1, #candidates do
    local candidate = candidates[i]
    local similarity = strOps.jaroWinklerSim(keyword, candidate)
    
    if similarity >= threshold then
        table.insert(results, {
            text = candidate,
            similarity = similarity
        })
    end
end

-- 按相似度排序（降序）
table.sort(results, function(a, b)
    return a.similarity > b.similarity
end)

-- 显示结果
for i = 1, #results do
    local result = results[i]
    print(string.format('  %d. "%s" (相似度: %.4f)', i, result.text, result.similarity))
end

注意事项 ​

  • 初始化要求：使用前必须先调用 UtilsMain.init(context) 完成插件初始化。

  • null 处理：大部分方法都支持 null 输入，会返回相应的默认值（如空字符串、false、0 等）。

  • JSON 格式：JSON 相关方法返回的都是 JSON 字符串，在 Lua 中需要使用 jsonLib.decode() 解析

  • 参数完整性：调用插件接口时，所有方法必须提供完整的参数，参数数量和类型必须与插件的接口完全一致。

  • 错误处理：JSON 处理方法在遇到非法输入时会返回默认值（如 "[]" 或 "{}"），不会抛出异常。
