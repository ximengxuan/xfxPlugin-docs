AppOps_App操作 ​

模块简介 ​

AppOps 封装了应用层常用操作，包括安装状态判断、前后台判断、重启/启动/退出、版本与包信息获取、Root 权限判断、首次安装判断、安装/卸载应用等。

⚠️ 不支持懒人精灵最高权限下运行(最高权限下获取不到Context)。

功能概览 ​

  • 上下文设置：设置 Context（setContext，与 UtilsMain.init 二选一）
  • UID 获取：当前 App 或指定包名的 UID（getAppUid）
  • 状态判断：是否安装、Debug、系统应用、前台、首次安装、Root 权限（isAppInstalled/isAppDebug/isAppSystem/isAppForeground/isFirstTimeInstalled/isAppRoot）
  • 启动/重启/退出：重启自身（relaunchApp）、启动 App（launchApp）、打开设置（launchAppDetailsSettings）、退出（exitApp）
  • 信息获取：包名、名称、路径、版本号/码（getAppPackageName/getAppName/getAppPath/getAppVersionName/getAppVersionCode）；已安装应用间 appName 与 packageName 互查（getPackageName）；指定应用完整信息（getAppInfo）
  • 安装/卸载：系统安装器安装（installApp）、系统卸载器卸载（uninstallApp）；静默安装应用（installAppSilent）、静默卸载应用（uninstallAppSilent，需要 Root 权限或系统权限）

API 接口列表 ​

setContext() ​

按需设置 Context（建议传入 applicationContext），在未调用 UtilsMain.init 时也可提前注入。

setContext()和 UtilsMain.init() 二选一调用即可。

参数名类型必填说明
contextContext是应用或 Activity 的 context，建议 applicationContext
返回值类型说明
void无

示例：

lua
import('com.nx.assist.lua.LuaEngine')

local softOps = UtilsMain.appOps()

local context = LuaEngine.getContext()
softOps.setContext(context)  -- 传入已有的 context

getPackageName(appName: String) ​

根据应用显示名称获取包名。 在本机已安装应用中按名称精确匹配（忽略大小写）；若存在多个同名应用，返回第一个匹配项。无需 Root 权限。

参数名类型必填说明

返回值类型说明
String包名；未找到返回空字符串

getAppInfo(packageName: String) ​

获取指定包名的 App 完整信息（JSON 格式）。

参数名类型必填说明
packageNameString是目标包名
返回值类型说明
StringJSON 字符串，包含 packageName、name、packagePath、versionName、versionCode、minSdkVersion、targetSdkVersion、isSystem 字段；失败返回空字符串

JSON 字段说明：

字段类型说明
packageNameString包名
nameString应用显示名称
packagePathStringAPK 文件路径
versionNameString版本名
versionCodeInt版本码
minSdkVersionInt最低 SDK 版本（Android 7.0+，否则返回 -1）
targetSdkVersionInt目标 SDK 版本
isSystemBoolean是否为系统应用

示例：

lua
local softOps = UtilsMain.appOps()

-- 获取指定应用信息
local json = softOps.getAppInfo("com.tencent.mm")
if json ~= "" then
    local info = jsonLib.decode(json)
    print("包名: " .. info.packageName)
    print("名称: " .. info.name)
    print("路径: " .. info.packagePath)
    print("版本名: " .. info.versionName)
    print("版本码: " .. tostring(info.versionCode))
    print("最低 SDK: " .. tostring(info.minSdkVersion))
    print("目标 SDK: " .. tostring(info.targetSdkVersion))
    print("是否系统应用: " .. tostring(info.isSystem))
end

print('========== App 操作示例 ==========')

local appPath = softOps.getAppPath()
local versionName = softOps.getAppVersionName()
local versionCode = softOps.getAppVersionCode()
local uid = softOps.getAppUid()

print('包名: ' .. packageName)
print('名称: ' .. appName)
print('路径: ' .. appPath)

print('版本码: ' .. tostring(versionCode))
print('UID: ' .. tostring(uid))

local isSystem = softOps.isAppSystem()
local isForeground = softOps.isAppForeground()

print('是否 Debug 版本: ' .. tostring(isDebug))
print('是否系统应用: ' .. tostring(isSystem))
print('是否在前台: ' .. tostring(isForeground))

-- ============================================
-- 3. 检查其他 App 是否安装
-- ============================================
print('\n--- 3. 检查其他 App 是否安装 ---')

-- 检查微信是否安装
local wechatPkg = 'com.tencent.mm'
local wechatInstalled = softOps.isAppInstalled(wechatPkg)
print('微信是否安装: ' .. tostring(wechatInstalled))

-- 检查 QQ 是否安装
local qqPkg = 'com.tencent.mobileqq'
local qqInstalled = softOps.isAppInstalled(qqPkg)
print('QQ 是否安装: ' .. tostring(qqInstalled))

-- 检查不存在的应用
local fakePkg = 'com.example.fake'
local fakeInstalled = softOps.isAppInstalled(fakePkg)
print('不存在的应用是否安装: ' .. tostring(fakeInstalled))


-- ============================================
-- 5. 获取其他 App 的 UID
-- ============================================
print('\n--- 5. 获取其他 App 的 UID ---')

if wechatInstalled then
    local wechatUid = softOps.getAppUid(wechatPkg)
    print('微信 UID: ' .. tostring(wechatUid))
end

-- ============================================
-- 6. 启动其他 App
-- ============================================
print('\n--- 6. 启动其他 App ---')

-- 注意：启动其他应用需要相应权限，请谨慎使用
if wechatInstalled then
    print('准备启动微信...')
    softOps.launchApp(wechatPkg)
    sleep(5000)
    
    -- -- 检查是否成功启动
    -- local running = softOps.isAppRunning(wechatPkg)
    -- print('微信是否运行: ' .. tostring(running))
else
    print('微信未安装，无法启动')
end

-- ============================================
-- 7. 打开 App 设置页面
-- ============================================
print('\n--- 7. 打开 App 设置页面 ---')

if wechatInstalled then
    print('准备打开微信设置页面...')
    softOps.launchAppDetailsSettings(wechatPkg)
    print('已打开微信设置页面')
    sleep(5000)
end

-- ============================================
-- 8. 重新启动当前 App
-- ============================================
print('\n--- 8. 重新启动当前 App ---')

-- 注意：此操作会重启当前应用，请谨慎使用
print('注意：重新启动应用会终止当前脚本执行')

-- 9. 关闭当前 App
-- ============================================

-- 注意：此操作会关闭当前应用，请谨慎使用

-- 检查应用是否安装

    print('目标应用已安装')

注意事项 ​

  • 初始化要求：使用前需先调用 xxxOps.setContext(context)或UtilsMain.init(context) 初始化上下文。
  • 权限：
    • 启动其他应用或打开设置页通常不需要额外权限
    • installApp() 在 Target API > 25 时需要 REQUEST_INSTALL_PACKAGES 权限
    • uninstallApp() 在 Target API > 25 时需要 REQUEST_DELETE_PACKAGES 权限
    • installAppSilent() 和 uninstallAppSilent() 需要 Root 权限或系统权限：
      • 无 Root 权限时必须持有 android:sharedUserId="android.uid.shell" 和相应的安装/卸载权限
      • 有 Root 权限时可直接使用
  • 静默安装/卸载：这些操作会直接执行，不会弹出用户确认对话框，请谨慎使用。
  • 普通安装/卸载：installApp() 和 uninstallApp() 会弹出系统安装/卸载界面，需用户手动确认。
