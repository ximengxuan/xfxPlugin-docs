TimeOps_时间操作 ​

模块简介 ​

TimeOps 是时间操作工具模块，封装了时间获取、格式化、字符串与毫秒/日期互转、当日判断以及时间范围判断等能力。

功能概览 ​

  • 时间获取：当前时间字符串、毫秒值、日期对象
  • 时间格式化与互转：毫秒 ↔ 字符串、字符串 ↔ 日期、日期 ↔ 字符串
  • 日期判断：判断指定时间是否为今天
  • 时间范围判断：支持跨天的时间段判断（如 22:00~06:00）
  • 时间差计算：计算两个时间戳之间的时间差，或将毫秒转换为可读的时间跨度格式

时区说明（重要） ​

TimeOps 所有「格式化 / 解析」类接口默认统一使用东八区（Asia/Shanghai），不再依赖设备系统当前时区。这样可以避免模拟器或设备时区被设置成 UTC / 其他时区时，出现时间偏差（例如比北京时间慢 8 小时）。

  • 默认时区固定为 Asia/Shanghai（北京时间），无需任何配置即可直接使用。
  • 如需切换时区（例如海外设备需要本地时间），可调用 setTimeZone(id) 全局设置；调用 resetTimeZone() 恢复默认东八区。
  • getTimeZone() 可查询当前生效的时区 ID。
  • 时间戳类接口（getNowMills、各类时间差计算）返回的是绝对时刻，与时区无关，不受影响。

注意：时区设置为全局生效。若多脚本 / 多任务并发使用，请在入口处统一设置一次即可，避免在运行时随意切换。

API 接口列表 ​

0. 时区设置 ​

setTimeZone(id: String?) ​

设置所有时间格式化 / 解析接口使用的时区。不传或传入空字符串时，恢复为默认东八区（Asia/Shanghai）。

参数名类型必填说明
idString?否时区 ID，如 "Asia/Shanghai"、"GMT+8"、"UTC" 等；null 或空串恢复默认东八区
返回值类型说明
无无返回值

示例：

lua
local timeOps = UtilsMain.timeOps()
timeOps.setTimeZone("Asia/Shanghai")  -- 固定北京时间（默认即此）
timeOps.setTimeZone("UTC")            -- 切换为 UTC
timeOps.setTimeZone("")               -- 恢复默认东八区

6. 时间范围判断 ​

isNowInTimeRange(startTime: String?, endTime: String?) ​

判断当前时间是否处于指定时间段内，支持跨天场景（如 22:00~06:00）。

参数名类型必填说明

返回值类型说明

示例：

print(timeOps.isNowInTimeRange("09:00", "18:00"))  -- 工作时间判断
print(timeOps.isNowInTimeRange("22:00", "06:00"))  -- 跨天判断

7. 时间差计算 ​

以下方法用于计算时间差或格式化时间跨度。

getTimeSpan(startMillis: Long, endMillis: Long) ​

获取两个时间戳之间的时间差，以更简练的方式返回（N天N小时N分N秒）。若天、小时或分钟为0，则省略对应单位。

参数名类型必填说明
startMillisLong是开始时间戳（毫秒）
endMillisLong是结束时间戳（毫秒）
返回值类型说明
String时间差字符串，例如"1小时2分15秒"

示例：

lua
local timeOps = UtilsMain.timeOps()
local start = timeOps.getNowMills()
-- ... 一些操作 ...
local end = timeOps.getNowMills()
print(timeOps.getTimeSpan(start, end))  -- 输出: 1小时2分15秒

完整示例 ​

lua
import('com.nx.assist.lua.LuaEngine')

local loader = LuaEngine.loadApk('xfxPlugin-release.apk')
if not loader then error('Failed to load APK') end

local context = LuaEngine.getContext()
local UtilsMain = loader.loadClass('com.xfx.plugin.UtilsMain')
if not UtilsMain then error('Failed to load Class') end

-- 初始化插件
UtilsMain.init(context)

-- 获取时间工具
local timeOps = UtilsMain.timeOps()

print("=== 时间获取 ===")
print("getNowString(): " .. timeOps.getNowString())
print("getNowString('yyyy/MM/dd'): " .. tostring(timeOps.getNowString("yyyy/MM/dd")))
print("getNowMills(): " .. tostring(timeOps.getNowMills()))

print("\\n=== 毫秒转字符串 ===")
print("millisToString(0): " .. tostring(timeOps.millisToString(0)))
print("millisToString(0, 'yyyy/MM/dd'): " .. tostring(timeOps.millisToString(0, "yyyy/MM/dd")))

print("\\n=== 字符串转毫秒 ===")
print("stringToMillis('2025-12-08 12:00:00'): " .. tostring(timeOps.stringToMillis("2025-12-08 12:00:00")))
print("stringToMillis('2025/12/08', 'yyyy/MM/dd'): " .. tostring(timeOps.stringToMillis("2025/12/08", "yyyy/MM/dd")))

print("\\n=== 字符串与日期互转 ===")
local dateObj = timeOps.stringToDate("2025-12-08 12:00:00")
print("stringToDate(...): " .. tostring(dateObj))
print("dateToString(dateObj): " .. tostring(timeOps.dateToString(dateObj)))
print("dateToString(dateObj, 'yyyy/MM/dd'): " .. tostring(timeOps.dateToString(dateObj, "yyyy/MM/dd")))

print("\\n=== 今日判断 ===")
print("isToday(now string): " .. tostring(timeOps.isToday(timeOps.getNowString())))
print("isToday(now millis): " .. tostring(timeOps.isToday(timeOps.getNowMills())))

print("\\n=== 时间范围判断 ===")
print("isNowInTimeRange('09:00','18:00'): " .. tostring(timeOps.isNowInTimeRange("09:00", "18:00")))
print("isNowInTimeRange('22:00','06:00'): " .. tostring(timeOps.isNowInTimeRange("22:00", "06:00")))

print("\\n=== 时间差计算 ===")
local startMillis = timeOps.getNowMills()
-- 模拟经过一段时间
local endMillis = startMillis + 7323000  -- 2小时2分3秒
print("getTimeSpan: " .. timeOps.getTimeSpan(startMillis, endMillis))  -- 输出: 2小时2分3秒
print("millisToTimeSpan(3600000): " .. timeOps.millisToTimeSpan(3600000))  -- 输出: 1小时
print("millisToTimeSpan(7323000): " .. timeOps.millisToTimeSpan(7323000))  -- 输出: 2小时2分3秒
print("getMinutesBetween: " .. tostring(timeOps.getMinutesBetween(startMillis, endMillis)))  -- 输出: 122（分钟）
print("getSecondsBetween: " .. tostring(timeOps.getSecondsBetween(startMillis, endMillis)))  -- 输出: 7323（秒）

注意事项 ​

  • 格式说明：未显式传入 pattern 时使用默认格式 yyyy-MM-dd HH:mm:ss；传入 pattern 需符合 SimpleDateFormat 规范。
  • null 处理：部分方法支持 null 输入并返回 null 或 false，请在 Lua/调用侧做好判空。
  • 时间范围格式：isNowInTimeRange 仅接受 HH:MM 格式，格式错误会直接返回 false；支持跨天判断（起始时间大于结束时间视为跨天）。
  • 时间差计算：getTimeSpan 会自动计算两个时间戳的绝对值差；millisToTimeSpan 会将毫秒数转换为可读的时间跨度格式，若某个单位为0则省略（但秒数即使为0也会显示）。
  • 时区：所有格式化 / 解析接口默认使用东八区（Asia/Shanghai），与设备系统当前时区无关；可通过 setTimeZone 切换、resetTimeZone 恢复、getTimeZone 查询。时间戳类接口（getNowMills 及时间差计算）为绝对时刻，不受影响。注意 TimeOps 是全局单例，时区设置为全局生效。
