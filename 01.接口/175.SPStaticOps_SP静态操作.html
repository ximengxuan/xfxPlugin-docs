SPStaticOps_SP静态操作 ​

模块简介 ​

SPStaticOps 直接使用默认 SP 文件进行字符串、整数、长整型、布尔的存取，并支持设置默认文件、移除与清空。

SharedPreferences 是Android平台上一个轻量级的存储类，主要用于保存应用的一些常用配置。它通过键值对的形式将数据保存在XML文件中，支持的数据类型包括String、int、long、float和boolean。

⚠️ 不支持懒人精灵最高权限下运行(最高权限下获取不到Context)。

功能概览 ​

  • 默认文件配置：设置默认 SP 文件
  • 读写操作：String / Int / Long / Boolean 的写入与读取（使用 put/get 重载方法）
  • 数据管理：移除键、清空默认文件

API 接口列表 ​

1. 默认文件 ​

setInstance() ​

设置默认 SP 文件。

不调用这个接口时，使用框架默认文件；如需隔离数据，可先设置专用文件名。

参数名类型必填说明
spNameString是SP 文件名

示例：

lua
local spStatic = UtilsMain.spStaticOps()
spStatic.setInstance("default_sp")

2

2. 数据读写操作 ​

put() ​

写入值（重载方法，支持多种类型）。

根据 value 参数的类型自动选择对应的存储方法。支持 String、Int、Long、Boolean 类型。

参数名类型必填说明
keyString是键
valueString / Int / Long / Boolean是值（类型自动识别）

重载方法：

  • put(key: String, value: String) - 写入字符串
  • put(key: String, value: Int) - 写入整数
  • put(key: String, value: Long) - 写入长整数
  • put(key: String, value: Boolean) - 写入布尔值

示例：

lua
local spStatic = UtilsMain.spStaticOps()

-- 写入字符串
spStatic.put("token", "abc")

-- 写入整数
spStatic.put("age", 18)

-- 写入长整数
spStatic.put("ts", 1234567890)

-- 写入布尔值
spStatic.put("login", true)

get() ​

读取值（重载方法，根据默认值类型自动推断返回类型）。

根据 defaultValue 参数的类型自动选择对应的读取方法。返回类型与默认值类型一致。

参数名类型必填说明
keyString是键
defaultValueString / Int / Long / Boolean是默认值（类型决定返回类型）
返回值类型说明
与 defaultValue 类型相同读取到的值，如果键不存在则返回默认值

重载方法：

  • get(key: String, defaultValue: String): String - 读取字符串
  • get(key: String, defaultValue: Int): Int - 读取整数
  • get(key: String, defaultValue: Long): Long - 读取长整数（defaultValue 可选，默认为 0）
  • get(key: String, defaultValue: Boolean): Boolean - 读取布尔值

示例：

lua
local spStatic = UtilsMain.spStaticOps()

-- 读取字符串
print(spStatic.get("token", ""))

-- 读取整数
print(spStatic.get("age", 0))

-- 读取长整数
print(spStatic.get("ts", 0))

-- 读取布尔值
print(spStatic.get("login", false))

完整示例 ​

lua
import('com.nx.assist.lua.LuaEngine')
local loader = LuaEngine.loadApk('xfxPlugin-release.apk'); assert(loader)
local context = LuaEngine.getContext()
local UtilsMain = loader.loadClass('com.xfx.plugin.UtilsMain'); assert(UtilsMain)
UtilsMain.init(context)

local spStatic = UtilsMain.spStaticOps()

spStatic.setInstance("default_sp")

spStatic.put("ts", 1234567890)
spStatic.put("token", "abc")
spStatic.put("age", 18)
spStatic.put("login", true)

print("ts:", spStatic.get("ts", 0))
print("token:", spStatic.get("token", ""))
print("age:", spStatic.get("age", 0))
print("login:", spStatic.get("login", false))
print(spStatic.getAll())

spStatic.remove("token")
spStatic.clear()
print(spStatic.getAll())

OOP 调用例子 ​

lua
-- ============================================
-- SharedPreferences 静态操作示例
-- 演示如何使用 XfxPlugin.lua 调用 SPStaticOps 进行数据存储
-- ============================================
import('com.nx.assist.lua.LuaEngine')

print('========== SharedPreferences 静态操作示例 ==========')

-- 加载 XfxPlugin.lua 类
local xfxModule = require('lib/XfxPlugin')

-- 创建 XFX 对象实例
local XFX = xfxModule:new({
    apkName = 'xfxPlugin-release.apk',  -- APK 文件名
})

XFX:setContext(LuaEngine.getContext())  -- 设置全局上下文

-- 获取 SPStaticOps 对象
local SPStaticOps = XFX:getOps('spStaticOps')
if not SPStaticOps then
    print('无法获取 SPStaticOps 对象')
    return
end

print('开始 SharedPreferences 静态操作示例...')

-- ============================================
-- 1. 设置默认 SP 文件名（可选）
-- ============================================
print('\n--- 1. 设置默认 SP 文件名 ---')

SPStaticOps.setInstance('my_app_config')
print('已设置默认 SP 文件名为: my_app_config')

-- ============================================
-- 2. 存储数据
-- ============================================
print('\n--- 2. 存储数据 ---')

-- 存储字符串
SPStaticOps.put('user_name', 'feiyun')
print('已存储用户名: feiyun')

-- 存储整数
SPStaticOps.put('login_count', 3)
print('已存储登录次数: 3')

-- 存储长整数
local timestamp = os.time() * 1000
SPStaticOps.put('last_login_time', timestamp)
print('已存储最后登录时间: ' .. tostring(timestamp))

-- 存储布尔值
SPStaticOps.put('is_vip', true)
print('已存储 VIP 标志: true')

-- ============================================
-- 3. 读取数据
-- ============================================
print('\n--- 3. 读取数据 ---')

local name = SPStaticOps.get('user_name', 'guest')
print('读取用户名: ' .. name)

local count = SPStaticOps.get('login_count', 0)
print('读取登录次数: ' .. tostring(count))

local lastTime = SPStaticOps.get('last_login_time', 0)
print('读取最后登录时间: ' .. tostring(lastTime))

local isVip = SPStaticOps.get('is_vip', false)
print('读取 VIP 标志: ' .. tostring(isVip))

-- ============================================
-- 4. 数据管理
-- ============================================
print('\n--- 4. 数据管理 ---')

-- 移除指定键
SPStaticOps.remove('user_name')
print('已移除用户名')

-- 清除所有数据
SPStaticOps.clear()
print('已清除所有数据')

print('\n========== SharedPreferences 静态操作示例完成 ==========')

注意事项 ​

  1. 初始化要求：使用前需先调用 xxxOps.setContext(context)或UtilsMain.init(context) 初始化上下文。

  2. 默认 SP 文件：

    • 使用前建议先设置默认 SP 文件名
    • 如果不设置，使用系统默认文件
    • 所有操作都使用同一个默认文件

  3. 与 SPOps 的区别：

    • SPStaticOps 使用默认文件，不需要每次指定文件名；
    • SPOps 需要每次指定文件名，支持多文件
    • 根据需求选择合适的工具
