NetworkOps_网络信息 ​

模块简介 ​

NetworkOps 提供网络状态检测、WiFi/移动数据开关、Ping 可用性检测、IP/运营商/网络类型查询等功能。

功能概览 ​

  • 上下文设置：设置 Context（setContext，与 UtilsMain.init 二选一）
  • 设置页：打开系统无线网络设置页
  • 连通性：网络是否连接、Ping 可用性（默认或指定 IP/域名）
  • 移动数据：获取是否开启、当前是否移动数据、是否 4G
  • WiFi：开关 WiFi、WiFi 是否连接/可用
  • 信息查询：运营商名、网络类型名、本机/公网 IP 地址、域名解析

API 接口列表 ​

setContext() ​

按需设置 Context（建议传入 applicationContext），在未调用 UtilsMain.init 时也可提前注入。

setContext()和 UtilsMain.init() 二选一调用即可。

参数名类型必填说明
contextContext是应用或 Activity 的 context，建议 applicationContext
返回值类型说明
void无

示例：

lua
import('com.nx.assist.lua.LuaEngine')

local netOps = UtilsMain.networkOps()

local context = LuaEngine.getContext()
netOps.setContext(context)  -- 传入已有的 context

3. WiFi 操作 ​

getWifiEnabled() ​

WiFi 是否开启。

返回值类型说明
Boolean开启返回 true

setWifiEnabled(enabled: Boolean) ​

设置 WiFi 开关。可能需要在弹出卡片中二次确认。

参数名类型必填说明
enabledBoolean是true 开启，false 关闭
返回值类型说明
void无返回值

4. 网络信息 ​

getNetworkOperatorName() ​

获取当前网络运营商名称。

返回值类型说明
String运营商名称

getNetworkTypeName() ​

获取当前网络类型名称。

返回值类型说明
String网络类型名

getIPAddress(useIPv4: Boolean) ​

获取本机 IP 地址。

参数名类型必填说明
useIPv4Boolean是true 取 IPv4，false 取 IPv6
返回值类型说明
String?IP 地址，失败返回 null

getPublicIPAddress(timeoutMillis: Int) ​

通过外网服务查询公网 IP 地址（需联网）。会依次尝试多个查询服务，任一成功即返回。

参数名类型必填说明
timeoutMillisInt是单次 HTTP 请求超时时间（毫秒），需大于 0
返回值类型说明
String?公网 IPv4 地址，失败返回 null

示例：

lua
local netOps = UtilsMain.networkOps()
print(netOps.getPublicIPAddress(10000))

完整示例 ​

lua
import('com.nx.assist.lua.LuaEngine')

-- 加载 APK 文件
local loader = LuaEngine.loadApk('xfxPlugin-release.apk')
if not loader then
    error('Failed to load APK')
end

-- 获取上下文
local context = LuaEngine.getContext()

-- 加载插件主类
local UtilsMain = loader.loadClass('com.xfx.plugin.UtilsMain')
if not UtilsMain then
    error('Failed to load Class')
end

-- 初始化插件（必须在使用任何工具类之前调用）
-- UtilsMain.init(context)

local netOps = UtilsMain.networkOps()

-- 以下方法无需上下文环境
print("ping default:", netOps.isAvailableByPing())
print("ping baidu :", netOps.isAvailableByPing("www.baidu.com"))
print("ip v4:", netOps.getIPAddress(true))
print("public ip:", netOps.getPublicIPAddress(10000))

netOps.setContext(context)  -- 传入已有的 context

-- 以下方法依赖上下文环境
print("connected:", netOps.isConnected())
print("wifi on:", netOps.getWifiEnabled())
print("wifi connected:", netOps.isWifiConnected())

print("mobile:", netOps.isMobileData(), "4g:", netOps.is4G())
print("operator:", netOps.getNetworkOperatorName(), "type:", netOps.getNetworkTypeName())
netOps.setWifiEnabled(true)

OOP 调用例子 ​

lua
-- ============================================
-- 网络操作示例
-- 演示如何使用 XfxPlugin.lua 调用 netOps 进行网络管理
-- ============================================
import('com.nx.assist.lua.LuaEngine')

print('========== 网络操作示例 ==========')

-- 加载 XfxPlugin.lua 类
local xfxModule = require('lib/XfxPlugin')

-- 创建 XFX 对象实例
local XFX = xfxModule:new({
    apkName = 'xfxPlugin-release.apk',  -- APK 文件名
})

-- 获取 netOps 对象
local netOps = XFX:getOps('networkOps')
if not netOps then
    error('无法获取 netOps 对象')
end

netOps.setContext(LuaEngine.getContext())  -- 传入已有的 context

print('开始网络操作示例...')

-- ============================================
-- 1. 检测网络连接状态
-- ============================================
print('\n--- 1. 检测网络连接状态 ---')

local isConnected = netOps.isConnected()
print('是否有网络连接: ' .. tostring(isConnected))

local isAvailable = netOps.isAvailableByPing()
print('网络是否可用（ping）: ' .. tostring(isAvailable))

local isAvailable2 = netOps.isAvailableByPing('8.8.8.8')
print('网络是否可用（ping 8.8.8.8）: ' .. tostring(isAvailable2))

-- ============================================
-- 2. WiFi 状态检测和管理
-- ============================================
print('\n--- 2. WiFi 状态检测和管理 ---')

local wifiEnabled = netOps.getWifiEnabled()
print('WiFi 是否已开启: ' .. tostring(wifiEnabled))

local wifiConnected = netOps.isWifiConnected()
print('WiFi 是否已连接: ' .. tostring(wifiConnected))


-- 开启 WiFi（如果需要）
if not wifiEnabled then
    print('准备开启 WiFi...')
    netOps.setWifiEnabled(true)
    sleep(2000)
    print('WiFi 已开启')
end

-- ============================================
-- 3. 移动数据检测
-- ============================================
print('\n--- 3. 移动数据检测 ---')

local mobileDataEnabled = netOps.getMobileDataEnabled()
print('移动数据是否开启: ' .. tostring(mobileDataEnabled))

local isMobile = netOps.isMobileData()
print('当前网络是否为移动数据: ' .. tostring(isMobile))

local is4G = netOps.is4G()
print('当前网络是否为 4G: ' .. tostring(is4G))

-- ============================================
-- 4. 获取网络信息
-- ============================================
print('\n--- 4. 获取网络信息 ---')

local operatorName = netOps.getNetworkOperatorName()
print('运营商名称: ' .. operatorName)

local networkType = netOps.getNetworkTypeName()
print('网络类型: ' .. networkType)

local ipv4 = netOps.getIPAddress(true)
print('IPv4 地址: ' .. tostring(ipv4))

local publicIp = netOps.getPublicIPAddress(10000)
print('公网 IP 地址: ' .. tostring(publicIp))

local ipv6 = netOps.getIPAddress(false)
print('IPv6 地址: ' .. tostring(ipv6))

-- ============================================
-- 5. 域名解析
-- ============================================
print('\n--- 5. 域名解析 ---')

local domain = 'www.baidu.com'
local ip = netOps.getDomainAddress(domain)
print('域名: ' .. domain)
print('IP 地址: ' .. tostring(ip))

-- ============================================
-- 6. 打开网络设置
-- ============================================
print('\n--- 6. 打开网络设置 ---')

-- 打开无线网络设置页面/连接与共享
netOps.openWirelessSettings()
-- print('已打开网络设置页面')

print('\n========== 网络操作示例完成 ==========')

注意事项 ​

  • 初始化要求：使用前请先调用 UtilsMain.init(context) 或 NetworkOps.setContext(context)。
  • 权限：
    • 部分 ROM 开关移动数据可能需要系统权限
    • WiFi 开关在新系统可能受限制
  • Ping 可用性：依赖系统 ping，可受网络/权限影响。
  • 公网 IP 查询：依赖外网 HTTP 服务（api.ipify.org、ifconfig.me、ip.3322.net、myip.ipip.net），网络不可用或服务异常时返回 null。
