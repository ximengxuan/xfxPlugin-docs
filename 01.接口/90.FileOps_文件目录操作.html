FileOps_文件目录操作 ​

模块简介 ​

FileOps 提供常用文件/目录操作：创建、读写、重命名、复制/移动、删除、大小/行数/MD5、路径工具，以及目录/文件信息的 JSON 列举。

功能概览 ​

  • 基础读写：读取/写入字符串/字节数组，创建文件/目录，追加内容
  • 属性与信息：存在性、文件/目录判断，名称/扩展名/行数/大小/MD5/文件信息
  • 操作：重命名、复制、移动、删除（文件/目录/目录内全部/按扩展名删除）
  • 路径与子目录：尾斜杠修饰、路径连接、子目录判断、是否有子目录、目录是否为空
  • 目录列举：按扩展名或直接子项输出 JSON

API 接口列表 ​

1. 基础读写与创建 ​

create() ​

根据路径自动创建文件或目录（已存在也返回 true）。 优先检查路径是否已存在且是目录，如果是则保持为目录。 否则根据路径是否有扩展名来判断创建文件还是目录。

参数名类型必填说明
pathString?是文件或目录路径
返回值类型说明
Boolean创建或已存在返回 true

示例：

lua
local FileOps = UtilsMain.fileOps()
print(FileOps.create("/sdcard/demo"))
print(FileOps.create("/sdcard/demo/a.txt"))

OOP 调用例子 ​

lua
-- ============================================
-- 文件操作示例
-- 演示如何使用 XfxPlugin.lua 调用 FileOps 进行文件操作
-- ============================================

print('========== 文件操作示例 ==========')

-- 加载 XfxPlugin.lua 类
local xfxModule = require('lib/XfxPlugin')

-- 创建 XFX 对象实例
local XFX = xfxModule:new({
    apkName = 'xfxPlugin-release.apk',  -- APK 文件名
})

-- 获取 FileOps 对象
local FileOps = XFX:getOps('fileOps')
if not FileOps then
    print('无法获取 FileOps 对象')
    return
end

print('开始文件操作示例...')

-- ============================================
-- 1. 创建目录和文件
-- ============================================
print('\n--- 1. 创建目录和文件 ---')

local baseDir = '/sdcard/xfx_test/'
local success1 = FileOps.create(baseDir)
print('创建目录: ' .. tostring(success1))

local filePath = FileOps.ensurePath(baseDir) .. 'test.txt'
local success2 = FileOps.create(filePath)
print('创建文件: ' .. tostring(success2))

-- ============================================
-- 2. 写入和读取文件
-- ============================================
print('\n--- 2. 写入和读取文件 ---')

local content = '这是测试内容\n第二行内容\n第三行内容'
local writeSuccess = FileOps.write(filePath, content)
print('写入文件: ' .. tostring(writeSuccess))

local readContent = FileOps.read(filePath)
print('读取内容: ' .. tostring(readContent))

-- ============================================
-- 3. 获取文件信息
-- ============================================
print('\n--- 3. 获取文件信息 ---')

local fileName = FileOps.getName(filePath)
print('文件名: ' .. fileName)

local fileNameNoExt = FileOps.getName(filePath,false)
print('文件名（无扩展名）: ' .. fileNameNoExt)

local extension = FileOps.getExtension(filePath)
print('扩展名: ' .. extension)

local size = FileOps.getSize(filePath)
print('文件大小: ' .. tostring(size))

local lines = FileOps.getLines(filePath)
print('文件行数: ' .. tostring(lines))

-- ============================================
-- 4. 文件操作（复制、移动、删除）
-- ============================================
print('\n--- 4. 文件操作（复制、移动、删除） ---')

-- 复制文件
local destPath = baseDir .. 'test_copy.txt'
local copySuccess = FileOps.copy(filePath, destPath)
print('复制文件: ' .. tostring(copySuccess))

-- 移动文件
local movePath = baseDir .. 'test_moved.txt'
local moveSuccess = FileOps.move(destPath, movePath)
print('移动文件: ' .. tostring(moveSuccess))

-- 删除文件
local deleteSuccess = FileOps.delete(movePath)
print('删除文件: ' .. tostring(deleteSuccess))

-- ============================================
-- 5. 列出文件
-- ============================================
print('\n--- 5. 列出文件 ---')

-- 创建几个测试文件
FileOps.write(baseDir .. 'file1.txt', 'content1')
FileOps.write(baseDir .. 'file2.txt', 'content2')
FileOps.write(baseDir .. 'file3.log', 'content3')

-- 列出所有 .txt 文件
local txtFilesJson = FileOps.listDir(baseDir, 'txt')
print('TXT 文件列表: ' .. txtFilesJson)

-- 列出所有文件
local allFilesJson = FileOps.listDir(baseDir, '')
print('所有文件列表: ' .. allFilesJson)

-- ============================================
-- 6. 目录操作
-- ============================================
print('\n--- 6. 目录操作 ---')

-- 创建子目录
local subDir = baseDir .. 'subdir/'
FileOps.create(subDir)

-- 判断子目录是否存在
local hasSub = FileOps.isSubDirExists(baseDir, 'subdir')
print('子目录是否存在: ' .. tostring(hasSub))

-- 判断是否有子目录
local hasSubs = FileOps.hasSubDirs(baseDir)
print('是否有子目录: ' .. tostring(hasSubs))

-- 获取一级子目录
local childrenJson = FileOps.listChildDir(baseDir, true)
print('一级子目录和文件: ' .. childrenJson)

-- ============================================
-- 7. 文件 MD5 计算
-- ============================================
print('\n--- 7. 文件 MD5 计算 ---')

local md5 = FileOps.md5(filePath)
print('文件 MD5: ' .. tostring(md5))

-- ============================================
-- 8. 清理测试文件
-- ============================================
print('\n--- 8. 清理测试文件 ---')

FileOps.clearDir(baseDir)
print('已清空测试目录')

FileOps.delete(baseDir)
print('已删除测试目录')
print(FileOps.isExists(baseDir))

print('\n========== 文件操作示例完成 ==========')

示例 5：文件完整性校验 ​

lua
local xfxModule = require('lib/XfxPlugin')
local XFX = xfxModule:new({apkName = 'xfxPlugin-release.apk'})

local FileOps = XFX:getOps('fileOps')
local EncryptOps = XFX:getOps('encryptOps')

if not FileOps or not EncryptOps then
    print('无法获取必要的 Ops 对象')
    return
end

local filePath = '/sdcard/important_file.txt'
local md5File = filePath .. '.md5'

-- 计算文件 MD5
if FileOps.isExists(filePath) then
    local currentMD5 = FileOps.md5(filePath)
    print('文件 MD5: ' .. currentMD5)

    -- 保存 MD5
    FileOps.write(md5File, currentMD5)

    -- 验证文件完整性
    sleep(1000)
    local newMD5 = FileOps.md5(filePath)
    local storedMD5 = FileOps.read(md5File)

    if newMD5 == storedMD5 then
        print('文件完整性验证通过')
    else
        print('警告：文件可能已被修改！')
    end
else
    print('文件不存在')
end

注意事项 ​

  1. 文件路径：

    • 使用绝对路径，如 "/sdcard/file.txt"
    • 确保路径中的文件和目录存在
    • 路径区分大小写

  2. 权限要求：

    • 读写外部存储需要相应权限
    • 某些系统目录可能需要特殊权限
    • 建议使用应用私有目录或 SD 卡

  3. 文件编码：

    • 默认使用 UTF-8 编码
    • 读取文件时注意编码问题
    • 二进制文件需要使用其他方法

  4. 文件删除：

    • 删除操作不可恢复
    • 删除前建议先备份重要文件
    • 删除目录会删除目录下所有内容

  5. JSON 格式：

    • listDir() 返回 JSON 数组字符串
    • 每个元素包含：fileName, filePath, lastModified, size
    • 需要使用 jsonLib 解析 JSON

  6. 目录操作：

    • 目录路径建议以 "/" 结尾
    • 可以使用 ensurePath() 确保路径格式正确

  7. 文件 MD5：

    • 大文件 MD5 计算可能需要较长时间
    • 建议在后台线程计算
    • MD5 用于文件完整性校验

  8. 错误处理：

    • 文件不存在时某些操作可能返回 false 或 null
    • 建议使用 isExists() 先检查
    • Lua可使用 pcall() 捕获可能的异常
