## 本地构建与测试

1、新电脑需要安装Nodejs，配置环境变量后，重启Cursor编辑器。

[安装Nodejs及设置环境变量（Windows平台）](http://www.feiyunjs.com/4975.html)

[安装Nodejs及设置环境变量（Mac平台）](http://www.feiyunjs.com/2069.html)


2、安装依赖：
```
npm install
```

[参考资料](https://vitepress.dev/zh/guide/deploy)

3、运行以下命令来构建文档：
```
npm run docs:build
```

> 如果构建报错，可将报错日志放在豆包中进行分析。

4、运行以下命令可以在本地预览：
```
npm run docs:preview
```

## 部署到1panel

可使用腾讯云在线的工具连接服务器，查看[1Panel](https://1panel.cn/docs/v2/installation/cli/)的管理地址。
查看后台地址和用户信息：
> 1pctl user-info


1、创建静态网站（需设置域名和目录名），目录名为`xfx.jidongyun.net`
2、将`docs/.vitepress/dist`目录上传到1panel的静态网站目录中。目录地址为`/opt/1panel/1panel/www/sites/xfx.jidongyun.net/index/`
3、设置运行目录为`/dist`，保存并重载
4、接下来就可以通过域名来访问构建好的静态文档
5、如果需要更新文档，只需要重新上传`/VitePress/docs/.vitepress/dist`目录即可

## 部署到宝塔

在宝塔添加php项目。根目录指向`docs/.vitepress/dist`，php版本选择`纯静态`。

## 常见问题

### 删除.user.ini文件

使用 chattr 命令移除文件的 i 属性。

```
chattr -i .user.ini
```

移除属性后，即可使用 rm 命令删除 .user.ini 文件。

```
rm -rf .user.ini
```
