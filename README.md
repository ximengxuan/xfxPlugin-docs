# xfxPlugin 小飞侠插件文档

> 安卓脚本开发增强接口文档

基于 [VitePress](https://vitepress.dev/) + [Teek 主题](https://github.com/Kele-Bingtang/vitepress-theme-teek) 构建。

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run docs:dev

# 构建
npm run docs:build

# 预览构建结果
npm run docs:preview
```
