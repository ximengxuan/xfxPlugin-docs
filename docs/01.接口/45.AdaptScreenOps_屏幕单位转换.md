## 模块简介

`AdaptScreenOps` 提供屏幕适配相关的基础换算和单位转换功能，支持 pt 与 px 互转。

## 功能概览

- **单位换算**：pt ↔ px
- **单位转换**：dp ↔ px，sp ↔ px


## API 接口列表

### 1. 单位换算

#### pt2Px(ptValue: Float)

将 pt 值转换为 px。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| ptValue | Float | 是 | pt 值 |

| 返回值类型 | 说明 |
|-----------|------|
| Int | 对应的 px 值 |

**示例：**
```lua
local adapOps = UtilsMain.adaptScreenOps()
print(adapOps.pt2Px(10.0))  -- 输出示例：15
```

#### px2Pt(pxValue: Float)

将 px 值转换为 pt。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| pxValue | Float | 是 | px 值 |

| 返回值类型 | 说明 |
|-----------|------|
| Int | 对应的 pt 值 |

**示例：**
```lua
local adapOps = UtilsMain.adaptScreenOps()
print(adapOps.px2Pt(30.0))  -- 输出示例：20
```

## 完整示例

```lua
import('com.nx.assist.lua.LuaEngine')
local loader = LuaEngine.loadApk('xfxPlugin-release.apk'); assert(loader)
local context = LuaEngine.getContext()
local UtilsMain = loader.loadClass('com.xfx.plugin.UtilsMain'); assert(UtilsMain)
UtilsMain.init(context)

local adaptScreenOps = UtilsMain.adaptScreenOps()

print("dp2px:", adaptScreenOps.dp2px(10.0))
print("px2dp:", adaptScreenOps.px2dp(100.0))
print("sp2px:", adaptScreenOps.sp2px(12.0))
print("px2sp:", adaptScreenOps.px2sp(48.0))
```
