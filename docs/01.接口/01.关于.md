# XFX Plugin - 安卓工具类插件

## 📖 简介

XFX Plugin（小飞侠插件） 是一个**免费**的专为安卓脚本开发框架设计的功能丰富的工具类插件，提供了超过 400+ 的 API 接口，方便安卓脚本开发者使用。

### 主要特性

- 🚀 **丰富的工具类**：提供了大量常用工具类
- 📡 **实时通信**：集成 MQTT 客户端，支持实时消息通信
- ❤️ **悬浮窗日志**：由`雪落`大神提供的源码改进而来，支持在屏幕上实时输出运行日志
- 🗣️ **文本转语音**：支持 TTS 文本转语音功能
- 🎬 **视频处理**：集成 FFMPEG，支持视频处理、转换等操作
- 📱 **设备操作**：提供设备信息获取、屏幕操作、音量控制等功能
- 🔐 **加密解密**：支持多种加密算法，保障数据安全
- 🖼️ **图像处理**：支持图片压缩、二维码生成与识别等功能

## 信息处理

XFX Plugin 不收集、获取或上传任何个人信息。
