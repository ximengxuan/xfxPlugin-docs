## 模块简介

`QrCodeOps` 提供二维码的识别与生成能力，支持从文件或 Bitmap 解码，并可生成二维码返回 Bitmap 或直接保存文件。

## 功能概览

- **解码**：从文件路径或 Bitmap 读取二维码文本，支持返回附加格式信息
- **编码**：生成二维码 Bitmap，或保存为图片文件（支持尺寸/容错/颜色/格式）
- **便捷接口**：提供简化的尺寸传参重载

## API 接口列表

### 1. 解码

#### decodeFile()

从图片文件路径识别二维码内容。

> 识别前，请检查文件是否存在，不存在的文件会识别失败。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| imagePath | String | 是 | 图片文件路径 |

| 返回值类型 | 说明 |
|-----------|------|
| String? | 识别到的内容，失败返回 null |

**示例：**
```lua
local qrOps = UtilsMain.qrCodeOps()
local text = qrOps.decodeFile("/sdcard/qrOps.png")
print(text)
```

#### decodeBitmap()

从 Bitmap 识别二维码内容。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| bitmap | Bitmap | 是 | 需识别的 Bitmap |

| 返回值类型 | 说明 |
|-----------|------|
| String? | 识别到的内容，失败返回 null |

**示例：**
```lua
local qrOps = UtilsMain.qrCodeOps()
-- bitmap 需由上层获取，这里仅示例调用
local text = qrOps.decodeBitmap(bitmapObj)
print(text)
```

#### decodeFileInfo()

从文件识别二维码内容并返回格式信息。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| imagePath | String | 是 | 图片文件路径 |

| 返回值类型 | 说明 |
|-----------|------|
| String? | JSON 字符串：`{"text":"内容","format":"QR_CODE","success":true}`，失败返回 null |

**示例：**
```lua
local qrOps = UtilsMain.qrCodeOps()
local info = qrOps.decodeFileInfo("/sdcard/qrOps.png")
print(info)
-- {"text":"hello file","format":"QR_CODE","success":true}
```

#### decodeBitmapInfo()

从 Bitmap 识别二维码内容并返回格式信息。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| bitmap | Bitmap | 是 | 需识别的 Bitmap |

| 返回值类型 | 说明 |
|-----------|------|
| String? | JSON 字符串：`{"text":"内容","format":"QR_CODE","success":true}`，失败返回 null |

**示例：**
```lua
local qrOps = UtilsMain.qrCodeOps()
local info = qrOps.decodeBitmapInfo(bitmapObj)
print(info)
```

### 2. 生成二维码

#### encodeToBitmap()

生成二维码 Bitmap。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| content | String | 是 | 待编码内容 |
| width | Int | 否 | 宽度，默认 300 |
| height | Int | 否 | 高度，默认 300 |
| margin | Int | 否 | 边距模块数，默认 1 |
| errorCorrectionLevel | String | 否 | 容错级别 L/M/Q/H，默认 M |
| foregroundColor | Int | 否 | 前景色 0xRRGGBB，默认黑 |
| backgroundColor | Int | 否 | 背景色 0xRRGGBB，默认白 |

| 返回值类型 | 说明 |
|-----------|------|
| Bitmap? | 生成的二维码，失败返回 null |

**示例：**
```lua
local qrOps = UtilsMain.qrCodeOps()
local bmp = qrOps.encodeToBitmap("hello", 400, 400, 2, "H", 0x000000, 0xFFFFFF)
print(bmp)
```

#### encodeToFile()

生成二维码并保存到文件。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| content | String | 是 | 待编码内容 |
| outputPath | String | 是 | 输出文件路径 |
| width | Int | 否 | 宽度，默认 300 |
| height | Int | 否 | 高度，默认 300 |
| margin | Int | 否 | 边距模块数，默认 1 |
| errorCorrectionLevel | String | 否 | 容错级别 L/M/Q/H，默认 M |
| foregroundColor | Int | 否 | 前景色 0xRRGGBB，默认黑 |
| backgroundColor | Int | 否 | 背景色 0xRRGGBB，默认白 |
| format | String | 否 | 图片格式 PNG/JPEG/WEBP，默认 PNG |
| quality | Int | 否 | JPEG 质量 0-100，默认 90 |

| 返回值类型 | 说明 |
|-----------|------|
| Boolean | 保存成功返回 true |

**示例：**
```lua
local qrOps = UtilsMain.qrCodeOps()
local ok = qrOps.encodeToFile("hello", "/sdcard/qrOps.png", 300, 300, 1, "M", 0x000000, 0xFFFFFF, "PNG", 90)
print(ok)
```

### 3. 便捷重载

#### encodeToBitmap()

按指定尺寸生成二维码 Bitmap（宽高相同）。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| content | String | 是 | 待编码内容 |
| size | Int | 是 | 宽高像素 |

| 返回值类型 | 说明 |
|-----------|------|
| Bitmap? | 生成的二维码，失败返回 null |

**示例：**
```lua
local qrOps = UtilsMain.qrCodeOps()
local bmp = qrOps.encodeToBitmap("hello", 256)
print(bmp)
```

#### encodeToFile()

按指定尺寸生成二维码并保存。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| content | String | 是 | 待编码内容 |
| outputPath | String | 是 | 输出文件路径 |
| size | Int | 是 | 宽高像素 |

| 返回值类型 | 说明 |
|-----------|------|
| Boolean | 保存成功返回 true |

**示例：**
```lua
local qrOps = UtilsMain.qrCodeOps()
local ok = qrOps.encodeToFile("hello", "/sdcard/qr_small.png", 256)
print(ok)
```

## 完整示例(1)

```lua
import('com.nx.assist.lua.LuaEngine')
local loader = LuaEngine.loadApk('xfxPlugin-release.apk'); assert(loader)
local context = LuaEngine.getContext()
local UtilsMain = loader.loadClass('com.xfx.plugin.UtilsMain'); assert(UtilsMain)
UtilsMain.init(context)

local qrOps = UtilsMain.qrCodeOps()

-- 解码
print("decode file:", qrOps.decodeFile("/sdcard/test/qr_out.png"))
print("decode file info:", qrOps.decodeFileInfo("/sdcard/test/qr_out.png"))

-- 假设 bitmapObj 已由其他接口获得
-- print("decode bitmap:", qrOps.decodeBitmap(bitmapObj))
-- print("decode bitmap info:", qrOps.decodeBitmapInfo(bitmapObj))

-- 编码 Bitmap
local bmp = qrOps.encodeToBitmap("hello world", 400, 400, 2, "H", 0x000000, 0xFFFFFF)
print("bitmap:", bmp)

-- 编码保存文件
local ok1 = qrOps.encodeToFile("hello file", "/sdcard/test/qr_out.png", 300, 300, 1, "M", 0x000000, 0xFFFFFF, "PNG", 90)
print("save1:", ok1)

-- 便捷尺寸重载
local bmp2 = qrOps.encodeToBitmap("short", 256)
print("bitmap2:", bmp2)

local ok2 = qrOps.encodeToFile("short file", "/sdcard/test/qr_out_small.png", 256)
print("save2:", ok2)
```

## 完整示例(2)

```lua
local xfxModule = require('lib/XfxPlugin')
local XFX = xfxModule:new({apkName = 'xfxPlugin-release.apk'})

local qrOps = XFX:getOps('qrCodeOps')
if not qrOps then
    print('无法获取 qrOps 对象')
    return
end

-- ============================================
-- 生成二维码示例
-- ============================================

-- 1. 简略接口：生成二维码并保存到文件（只需内容和尺寸）
local success1 = qrOps.encodeToFile('https://www.example.com', '/sdcard/qrcode_simple.png', 300)
if success1 then
    print('简略接口生成成功：/sdcard/test/qrcode_simple.png')
else
    print('生成失败')
end

-- 2. 简略接口：生成二维码 Bitmap（只需内容和尺寸）
local bitmap1 = qrOps.encodeToBitmap('简略接口示例', 400)
if bitmap1 then
    print('简略接口 Bitmap 生成成功')
    local ImageOps = xfx:getOps('imageOps')
    if ImageOps then
        ImageOps.saveAsPng(bitmap1, '/sdcard/test/qrcode_simple_bitmap.png', true)
        print('从简略接口 Bitmap 保存：/sdcard/test/qrcode_simple_bitmap.png')
    end
end

-- 3. 完整接口：生成二维码（自定义尺寸和颜色）
local success2 =
    qrOps.encodeToFile(
    'Hello, World!', -- 内容
    '/sdcard/test/qrcode_custom.png', -- 输出路径
    500, -- 宽度（像素）
    500, -- 高度（像素）
    2, -- 边距（模块数）
    'H', -- 错误纠正级别：L/M/Q/H（H为最高）
    0xFF000000, -- 前景色（黑色，格式：0xRRGGBB）
    0xFFFFFFFF, -- 背景色（白色）
    'PNG', -- 格式：PNG/JPEG/WEBP
    100 -- 质量（仅对JPEG有效，0-100）
)
if success2 then
    print('自定义二维码生成成功：/sdcard/test/qrcode_custom.png')
end

-- 4. 生成彩色二维码
local success3 =
    qrOps.encodeToFile(
    '彩色二维码示例',
    '/sdcard/test/qrcode_color.png',
    400,
    400,
    1,
    'M',
    0xFF0066FF, -- 蓝色前景
    0xFFFFE6E6, -- 浅红色背景
    'PNG',
    100
)
if success3 then
    print('彩色二维码生成成功：/sdcard/test/qrcode_color.png')
end

-- 5. 生成二维码 Bitmap（用于进一步处理）
local bitmap =
    qrOps.encodeToBitmap(
    'Bitmap 二维码示例',
    300,
    300,
    2, -- 边距（模块数）
    'H', -- 错误纠正级别：L/M/Q/H（H为最高）
    0xFF000000, -- 前景色（黑色，格式：0xRRGGBB）
    0xFFFFFFFF -- 背景色（白色）
)
if bitmap then
    print('二维码 Bitmap 生成成功')
    -- 可以进一步使用 ImageOps 处理这个 Bitmap
    local ImageOps = xfx:getOps('imageOps')
    if ImageOps then
        ImageOps.saveAsPng(bitmap, '/sdcard/test/qrcode_from_bitmap.png', true)
        print('从 Bitmap 保存的二维码：/sdcard/test/qrcode_from_bitmap.png')
    end
else
    print('二维码 Bitmap 生成失败')
end

-- 6. 生成不同错误纠正级别的二维码
local ecLevels = {'L', 'M', 'Q', 'H'}
for i, level in ipairs(ecLevels) do
    local path = '/sdcard/test/qrcode_ec_' .. level .. '.png'
    local success =
        qrOps.encodeToFile(
        '错误纠正级别：' .. level,
        path,
        300,
        300,
        1,
        level,
        0xFF000000, -- 前景色（黑色，格式：0xRRGGBB）
        0xFFFFFFFF, -- 背景色（白色）
        'PNG', -- 格式：PNG/JPEG/WEBP
        100 -- 质量（仅对JPEG有效，0-100）
    )
    if success then
        print('错误纠正级别 ' .. level .. ' 的二维码已生成：' .. path)
    end
end

-- ============================================
-- 识别二维码示例
-- ============================================

print(fileExist('/sdcard/test/qrcode_custom.png'))

-- 从图片文件识别二维码
local result = qrOps.decodeFile('/sdcard/test/qrcode_custom.png')
if result then
    print('识别结果：' .. result)
else
    print('识别失败')
end

-- 获取详细信息（JSON格式）
local info = qrOps.decodeFileInfo('/sdcard/test/qrcode_custom.png')
if info then
    print('详细信息：' .. info)
end
```

## 注意事项

- **文件读写**：生成/保存二维码需确保有对应的存储读写权限与可写路径。
