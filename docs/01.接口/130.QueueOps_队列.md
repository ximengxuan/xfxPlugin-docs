## 模块简介

`QueueOps` 提供线程安全的队列管理，支持按 ID 持有多个队列，覆盖创建、入队/出队、查看队首/队尾、状态查询、清空/删除以及导出 JSON。

## 功能概览

- **队列管理**：创建/删除队列实例
- **入队出队**：入队、出队、查看队首与队尾
- **状态查询**：获取大小、判断是否为空
- **清理与导出**：清空队列、转为 JSON 数组字符串

## API 接口列表

### 1. 队列管理

#### create()

创建或获取队列实例（若已存在则直接返回 true）。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| id | String | 是 | 队列唯一标识 |

| 返回值类型 | 说明 |
|-----------|------|
| Boolean | 创建/获取成功返回 true |

**示例：**
```lua
local queueOps = UtilsMain.queueOps()
print(queueOps.create("q1"))
```

#### delete()

删除队列实例。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| id | String | 是 | 队列唯一标识 |

| 返回值类型 | 说明 |
|-----------|------|
| Boolean | 删除成功返回 true |

**示例：**
```lua
local queueOps = UtilsMain.queueOps()
print(queueOps.delete("q1"))
```

### 2. 入队出队

#### enqueue()

将元素添加到队尾。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| id | String | 是 | 队列唯一标识 |
| element | String | 否 | 元素内容，空则存入空串 |

| 返回值类型 | 说明 |
|-----------|------|
| Boolean | 入队成功返回 true |

**示例：**
```lua
local queueOps = UtilsMain.queueOps()
queueOps.create("q1")
print(queueOps.enqueue("q1", "item1"))
```

#### dequeue()

从队首移除并返回元素。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| id | String | 是 | 队列唯一标识 |

| 返回值类型 | 说明 |
|-----------|------|
| String? | 队首元素，队列空或失败返回 null |

**示例：**
```lua
local queueOps = UtilsMain.queueOps()
queueOps.create("q1")
queueOps.enqueue("q1", "item1")
print(queueOps.dequeue("q1"))
```

### 3. 查看元素

#### peek()

查看队首元素但不移除。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| id | String | 是 | 队列唯一标识 |

| 返回值类型 | 说明 |
|-----------|------|
| String? | 队首元素，队列空或失败返回 null |

**示例：**
```lua
local queueOps = UtilsMain.queueOps()
queueOps.create("q1")
queueOps.enqueue("q1", "item1")
print(queueOps.peek("q1"))
```

#### peekLast()

查看队尾元素但不移除。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| id | String | 是 | 队列唯一标识 |

| 返回值类型 | 说明 |
|-----------|------|
| String? | 队尾元素，队列空或失败返回 null |

**示例：**
```lua
local queueOps = UtilsMain.queueOps()
queueOps.create("q1")
queueOps.enqueue("q1", "item1")
queueOps.enqueue("q1", "item2")
print(queueOps.peekLast("q1"))
```

### 4. 状态查询

#### size()

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| id | String | 是 | 队列唯一标识 |

| 返回值类型 | 说明 |
|-----------|------|

**示例：**

#### isEmpty()

判断队列是否为空。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| id | String | 是 | 队列唯一标识 |

| 返回值类型 | 说明 |
|-----------|------|
| Boolean | 为空返回 true |

**示例：**
```lua
local queueOps = UtilsMain.queueOps()

```

### 5. 清理与导出

#### clear()

清空队列中的所有元素。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| id | String | 是 | 队列唯一标识 |

| 返回值类型 | 说明 |
|-----------|------|
| Boolean | 清空成功返回 true |

**示例：**
```lua
local queueOps = UtilsMain.queueOps()
queueOps.create("q1")
queueOps.enqueue("q1", "item1")
print(queueOps.clear("q1"))
```

#### toJsonArray()

将队列元素转为 JSON 数组字符串。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| id | String | 是 | 队列唯一标识 |

| 返回值类型 | 说明 |
|-----------|------|
| String | JSON 数组字符串，失败或不存在返回 "[]" |

**示例：**
```lua
local queueOps = UtilsMain.queueOps()
queueOps.create("q1")
queueOps.enqueue("q1", "item1")
queueOps.enqueue("q1", "item2")
print(queueOps.toJsonArray("q1"))
```

## 注意事项

- **ID 一致性**：所有操作需使用同一队列 ID；未创建的 ID 将导致操作失败。
- **并发安全**：内部使用 `ConcurrentHashMap` 与 `Queue`，适用于多线程场景，但调用方仍需按需同步业务逻辑。 

### 栈和队列区别

| 对比维度 | 栈（Stack） | 队列（Queue） |
| ---- | ---- | ---- |
| **存取规则** | 后进先出（LIFO），最后存入的最先取出 | 先进先出（FIFO），最先存入的最先取出 |
| **操作端** | 仅在**同一端**（栈顶）进行入栈、出栈 | 两端分离：队尾入队，队头出队 |
| **核心操作** | 入栈(push)、出栈(pop)、取栈顶(peek) | 入队(enqueue)、出队(dequeue)、取队首(peek)、取队尾(peekLast) |
| **形象比喻** | 手枪弹夹、叠放的盘子 | 排队购票、水管流水 |
| **典型应用** | 函数调用栈、表达式求值、括号匹配、回溯 | 消息队列、任务排队、广度优先搜索(BFS) |
| **遍历顺序** | 逆序访问存入元素 | 顺序访问存入元素 |
