## 模块简介

`ToastyOps` 基于悬浮窗模拟 Toast，提供更丰富的样式（成功/错误/警告/信息/普通/自定义）。与 `FloatLog` 的模拟 Toast 类似。

> ⚠️ 不支持懒人精灵「最高权限」运行（最高权限下获取不到 Context）。需要悬浮窗权限才能正常显示。

## 功能概览

- **上下文设置**：设置 Context（setContext，与 UtilsMain.init 二选一）
- **基础样式**：success / error / warn / info / normal
- **完全自定义**：颜色、位置、间距、字体大小
- **管理**：dismiss 主动关闭、错误码查询

## 错误码

| code | 含义 |
|------|------|
| 0 | ERR_OK，成功 |
| -1 | ERR_INVALID_PARAM，参数非法（空文本、时长<=0 等） |
| -2 | ERR_NO_CONTEXT，未设置可用 Context |
| -3 | ERR_ADD_VIEW_FAILED，窗口添加失败 |

## API 接口列表

### setContext()

按需设置 Context（建议传入 applicationContext），在未调用 `UtilsMain.init` 时也可提前注入。setContext 与 UtilsMain.init 二选一。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| context | Context | 是 | 应用或 Activity 的 context |

| 返回值类型 | 说明 |
|-----------|------|
| void | 无 |

### getLastErrorCode()

获取最近一次调用的错误码（见上表）。

| 返回值类型 | 说明 |
|-----------|------|
| Int | 错误码 |

### getLastErrorMsg()

获取最近一次调用的错误信息。

| 返回值类型 | 说明 |
|-----------|------|
| String? | 错误描述，可为空 |

### normal()

普通 Toast。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| message | String? | 是 | 文本 |
| duration | Long | 否 | 持续时间，毫秒 |

| 返回值类型 | 说明 |
|-----------|------|
| Int | 错误码 |

### success() / error() / warn() / info()

预设样式 Toast。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| message | String? | 是 | 文本 |
| duration | Long | 否 | 持续时间，毫秒 |

| 返回值类型 | 说明 |
|-----------|------|
| Int | 错误码 |

### custom()

完全自定义样式。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| message | String? | 是 | 文本 |
| duration | Long | 是 | 持续时间，毫秒 |
| gravityInt | Int | 是 | 位置：0=顶部，1=中间，2=底部 |
| backgroundColor | Int | 是 | 背景颜色（argb/int） |
| textColor | Int | 是 | 文字颜色 |
| textSize | Float | 是 | 文字大小（sp） |
| padding | Int | 是 | 内边距（px） |
| margin | Int | 是 | 外边距（px） |

| 返回值类型 | 说明 |
|-----------|------|
| Int | 错误码 |

### close()

主动关闭当前 Toast。

| 返回值类型 | 说明 |
|-----------|------|
| Int | 错误码 |

## 完整示例（Lua，显式传参）

```lua
import('com.nx.assist.lua.LuaEngine')

local loader = LuaEngine.loadApk('xfxPlugin-release.apk')
local UtilsMain = loader.loadClass('com.xfx.plugin.UtilsMain')

local tipOps = UtilsMain.toastOps()
local context = LuaEngine.getContext()
tipOps.setContext(context)  -- 建议设置

-- 普通（单参数，默认 2 秒）
tipOps.normal("普通Toast")
sleep(3000)

-- 普通（双参数，自定义时长）
tipOps.normal("自定义时长普通Toast", 3000)
sleep(3000)

-- 成功 / 错误 / 警告 / 信息（单参数，默认 2 秒）
tipOps.success("成功提示")
sleep(2000)

tipOps.error("错误提示")
sleep(2000)

tipOps.warn("警告提示")
sleep(2000)

tipOps.info("信息提示")
sleep(2000)

-- 成功 / 错误 / 警告 / 信息（双参数，自定义时长）
tipOps.success("成功提示", 5000)
sleep(5000)

tipOps.error("错误提示", 3000)
sleep(3000)

tipOps.warn("警告提示", 4000)
sleep(4000)

tipOps.info("信息提示", 2500)
sleep(2500)

-- 完全自定义
-- 位置：0=顶部，1=中间，2=底部
import('android.graphics.Color')
tipOps.custom(
    "自定义样式",
    3000,
    2,              -- gravityInt 底部
    0xAA000000,     -- backgroundColor
    Color.WHITE,    -- textColor
    16.0,           -- textSize
    16,             -- padding px
    50              -- margin px
)
sleep(3000)

-- 主动关闭
tipOps.close()

print("lastErrorCode", tipOps.getLastErrorCode())
print("lastErrorMsg", tipOps.getLastErrorMsg())
```

## 注意事项

- 使用前需调用 `setContext(context)` 或 `UtilsMain.init(context)` 初始化上下文。
- 需要悬浮窗权限，Android 6.0+ 无权限时可能无法显示。
- 所有方法均在 UI 线程执行，调用端无需再切线程。
