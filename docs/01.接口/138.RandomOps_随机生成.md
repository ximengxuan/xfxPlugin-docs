## 模块简介

`RandomOps` 提供随机虚拟数据生成能力。

当前版本主要开放 **昵称** 与 **中文/英文姓名** 生成，适用于游戏新建角色、批量注册、测试数据填充等场景。

类内已预留扩展点（互联网信息、语言文字、地区、教育、金融等），后续可按需继续开放更多随机内容生成方法。

## 功能概览

- **汉字网络昵称**：`chineseNickName`
- **英文网络昵称**：`englishNickName`
- **拼音网络昵称**：`pinyinNickName`
- **QQ 风格网名**：`qqNickName`
- **中文真实姓名**：`chineseName` / `maleChineseName` / `femaleChineseName`
- **按性别取姓名**：`chineseNameByGender`
- **英文姓名**：`englishName`
- **批量生成**：`chineseNickNames` / `chineseNames`（返回 JSON 数组字符串）

## API 接口列表

### 1. 昵称生成

#### chineseNickName()

生成随机汉字网络昵称（默认最长 6 个汉字）。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| 无 | - | - | 无参数 |

| 返回值类型 | 说明 |
|-----------|------|
| String | 随机汉字网络昵称 |

**示例：**
```lua
local randomOps = UtilsMain.randomOps()
print(randomOps.chineseNickName())
```

#### chineseNickName(maxLength)

生成随机汉字网络昵称，可指定最大汉字数。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| maxLength | Int | 是 | 最大汉字数，有效范围 1~32；传 0 或负数时使用默认值 6 |

| 返回值类型 | 说明 |
|-----------|------|
| String | 随机汉字网络昵称 |

**示例：**
```lua
local randomOps = UtilsMain.randomOps()
-- 适配王者荣耀 6 字昵称上限
print(randomOps.chineseNickName(6))
```

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|

| 返回值类型 | 说明 |
|-----------|------|

## 完整示例

### 某手游新建角色昵称

```lua
local randomOps = UtilsMain.randomOps()
local hashSetOps = UtilsMain.hashSetOps()

hashSetOps.create("usedNicknames")

--[[
 * 生成一个未使用过的游戏昵称
 * @param maxTry 最大重试次数
 * @return string|nil
]]
local function nextGameNickname(maxTry)
    maxTry = maxTry or 20
    for i = 1, maxTry do
        local nick = randomOps.chineseNickName(6)
        if not hashSetOps.contains("usedNicknames", nick) then
            hashSetOps.add("usedNicknames", nick)
            return nick
        end
    end
    return nil
end

local nickname = nextGameNickname(30)
if nickname then
    print("新角色昵称:", nickname)
else
    print("生成昵称失败，请增加重试次数")
end
```

### 批量生成候选昵称

```lua
local randomOps = UtilsMain.randomOps()

local json = randomOps.chineseNickNames(20, 6)
local names = jsonLib.decode(json)

print("候选昵称列表:")
for i, name in ipairs(names) do
    print(string.format("%02d. %s", i, name))
end
```

### 真实姓名 vs 网络昵称

```lua
local randomOps = UtilsMain.randomOps()

-- 偏真实玩家名
print("真实姓名:", randomOps.chineseName())

-- 偏网络化昵称
print("网络昵称:", randomOps.chineseNickName(6))
print("QQ风格:", randomOps.qqNickName())
```

## 注意事项

1. 所有方法参数均为 Lua 支持的基础类型（String、Int），批量结果以 **JSON 字符串** 返回。
2. `chineseNickName(6)` 仅控制生成长度上限，不保证一定能通过游戏敏感词校验，业务侧需自行重试或过滤。
3. 批量接口 `count` 最大为 100，超出会自动截断。
4. 与 `NumberOps.random()` 不同，`RandomOps` 专注于虚拟人物相关数据，二者互不冲突。
