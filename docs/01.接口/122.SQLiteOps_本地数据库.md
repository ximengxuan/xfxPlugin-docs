## 模块简介

`SQLiteOps` 提供本地 SQLite 操作，包含打开/关闭、CRUD、事务与简单异步写入。存储的文件会随着应用的卸载而清空。

> ⚠️ 不支持懒人精灵`最高权限`下运行(最高权限下获取不到Context)。

## 功能概览

- **上下文设置**：setContext 注入 Context（未调用 UtilsMain.init 时也可提前注入）
- **打开/关闭**：按数据库文件名管理，位于应用私有目录；open 打开、close 关闭
- **CRUD**：execSql、insert、update、delete、query（结果 JSON）
- **事务**：手动 begin/setSuccessful/end 或一次性 runInTransaction
- **异步**：execSqlAsync 简单线程 fire-and-forget
- **数据维护**：clear 清空所有用户表数据（保留表结构）、delete 删除数据库文件
- **参数规约**：所有方法必须按声明顺序显式传递全部参数；列表一律用 JSON 字符串

## 参数/数据格式约定

- `createSqlArrayJson`、`upgradeSqlArrayJson`、`argsJson`、`sqlArrayJson`：形如 `["SQL1","SQL2"]` 的 JSON 字符串
- `valuesJson`：形如 `{"col":"value","num":1,"flag":true}` 的 JSON 字符串
- `query` 返回值：JSON 数组字符串，每行是对象，例如 `[{"id":1,"title":"a"}]`

## API 接口列表


#### setContext()

按需设置 Context（建议传入 applicationContext），在未调用 `UtilsMain.init` 时也可提前注入。

> setContext()和 UtilsMain.init() 二选一调用即可。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| context | Context | 是 | 应用或 Activity 的 context，建议 applicationContext |

| 返回值类型 | 说明 |
|-----------|------|
| void | 无 |

**示例：**
```lua
import('com.nx.assist.lua.LuaEngine')

local dbOps = UtilsMain.sqliteOps()

local context = LuaEngine.getContext()
dbOps.setContext(context)  -- 传入已有的 context
```

### open()

打开或创建数据库。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| dbName | String | 是 | 数据库文件名（位于应用私有目录） |
| version | Int | 是 | 数据库版本 |
| createSqlArrayJson | String? | 否 | 创建表 SQL 的 JSON 数组字符串 |
| upgradeSqlArrayJson | String? | 否 | 升级 SQL 的 JSON 数组字符串 |

**返回：** Boolean

### close()

关闭数据库。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| dbName | String | 是 | 数据库文件名 |

**返回：** Boolean

### execSql()

执行非查询 SQL。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| dbName | String | 是 | 数据库文件名 |
| sql | String | 是 | SQL 语句 |
| argsJson | String? | 否 | 占位符参数 JSON 数组字符串 |

**返回：** Boolean

### execSqlAsync()

在新线程执行非查询 SQL（fire-and-forget）。

参数同 `execSql`。  
**返回：** Boolean（仅表示线程已提交）

### query()

执行查询并返回 JSON 字符串。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| dbName | String | 是 | 数据库文件名 |
| sql | String | 是 | 查询语句 |
| argsJson | String? | 否 | 占位符参数 JSON 数组字符串(SQL 中没有占位符 ? 的话可以不传) |

**返回：** JSON 字符串（数组）

```lua
-- SQL 中有占位符
local rowsJson = dbOps.query(dbName, "SELECT * FROM todo WHERE done=?",'[0]')
print(rowsJson)

-- 不传占位符参数
print(dbOps.query(dbName, "SELECT * FROM todo"))
```

### insert()

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| dbName | String | 是 | 数据库文件名 |
| table | String | 是 | 表名 |
| valuesJson | String | 是 | 列值 JSON 对象字符串 |

**返回：** Long（rowId，失败 -1）

```lua
print('ID：',db.insert("sample.db", "todo", jsonLib.encode({title = 'test', done = 0})))
```

### update()

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| dbName | String | 是 | 数据库文件名 |
| table | String | 是 | 表名 |
| valuesJson | String | 是 | 列值 JSON 对象字符串 |
| whereClause | String? | 否 | where 子句 |
| whereArgsJson | String? | 否 | where 占位符参数 JSON 数组 |

**返回：** Int（受影响行数，失败 -1）

### delete()

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| dbName | String | 是 | 数据库文件名 |
| table | String | 是 | 表名 |
| whereClause | String? | 否 | where 子句 |
| whereArgsJson | String? | 否 | where 占位符参数 JSON 数组 |

**返回：** Int（受影响行数，失败 -1）

### beginTransaction() / setTransactionSuccessful() / endTransaction()

按顺序配合使用。

| 方法 | 返回 | 说明 |
|------|------|------|
| beginTransaction(dbName) | Boolean | 开启事务 |
| setTransactionSuccessful(dbName) | Boolean | 标记成功 |
| endTransaction(dbName) | Boolean | 结束事务 |

### runInTransaction()

一次性事务执行多条 SQL。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| dbName | String | 是 | 数据库文件名 |
| sqlArrayJson | String | 是 | SQL 的 JSON 数组字符串 |

**返回：** Boolean

### clear()

清空数据库中所有用户表数据（保留表结构）。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| dbName | String | 是 | 数据库文件名 |

**返回：** Boolean

### delete()

删除数据库文件（会先关闭缓存连接）。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| dbName | String | 是 | 数据库文件名 |

**返回：** Boolean

## Lua 调用示例

```lua
import('com.nx.assist.lua.LuaEngine')
local loader  = LuaEngine.loadApk('xfxPlugin-release.apk')
local context = LuaEngine.getContext()
local UtilsMain    = loader.loadClass('com.xfx.plugin.UtilsMain')

-- -- 容错打印 init 结果
-- local ok, err = pcall(function() UtilsMain.init(context) end)
-- if not ok then
--     print('init error: ', err)
-- end

local dbOps = UtilsMain.sqliteOps()
dbOps.setContext(context)

local dbName = "sample.db"

-- 1) 打开/创建数据库并建表（SQL 用 JSON 数组字符串传入）
local createSqls = {
    "CREATE TABLE IF NOT EXISTS todo(id INTEGER PRIMARY KEY AUTOINCREMENT, title TEXT, done INTEGER DEFAULT 0)"
}
print(dbOps.open(dbName, 1, jsonLib.encode(createSqls), nil))

-- 2) 插入
print('ID：',dbOps.insert(dbName, "todo", jsonLib.encode({title = 'test', done = 0})))

-- 3) 查询（结果为 JSON 字符串）
local rowsJson = dbOps.query(dbName, "SELECT * FROM todo WHERE done=?",'[0]')
print(rowsJson)

-- 4) 事务执行多条 SQL
local txSqls = {
    "UPDATE todo SET done=1 WHERE id=1",
    "DELETE FROM todo WHERE done=1"
}
print('事务执行结果：', dbOps.runInTransaction(dbName, jsonLib.encode(txSqls)))

-- 不传占位符参数
print(dbOps.query(dbName, "SELECT * FROM todo"))

-- 5) 关闭
dbOps.close(dbName)
```

## OOP 调用例子

```lua
-- ============================================
-- SQLite 操作示例（OOP 封装方式）
-- 依赖 lib/XfxPlugin.lua 封装
-- ============================================
import('com.nx.assist.lua.LuaEngine')

local xfxModule = require('lib/XfxPlugin')
local XFX = xfxModule:new({
    apkName = 'xfxPlugin-release.apk',
    context = LuaEngine.getContext(),
})

-- 获取 sqliteOps
local dbOps = XFX:getOps('sqliteOps')
if not dbOps then
    error('无法获取 sqliteOps 对象')
end

local dbName = 'sample.db'
local createSqls = {
    "CREATE TABLE IF NOT EXISTS todo(id INTEGER PRIMARY KEY AUTOINCREMENT, title TEXT, done INTEGER DEFAULT 0)"
}

-- 打开并建表
print('open: ', dbOps.open(dbName, 1, jsonLib.encode(createSqls), nil))

-- 插入数据
print('insert id: ', dbOps.insert(dbName, 'todo', jsonLib.encode({title='task1', done=0})))

-- 查询（无占位符）
print('all rows: ', dbOps.query(dbName, 'SELECT * FROM todo'))

-- 查询（有占位符）
print('rows done=0: ', dbOps.query(dbName, 'SELECT * FROM todo WHERE done=?', '[0]'))

-- 关闭
dbOps.close(dbName)
```

## 实际应用案例

### 案例 1：待办列表增删改查

```lua
import('com.nx.assist.lua.LuaEngine')

local xfxModule = require('lib/XfxPlugin')
local XFX = xfxModule:new({
    apkName = 'xfxPlugin-release.apk',
    context = LuaEngine.getContext(),
})

local dbOps = XFX:getOps('sqliteOps')
local dbName = 'todo.db'
local createSqls = {
    "CREATE TABLE IF NOT EXISTS todo(id INTEGER PRIMARY KEY AUTOINCREMENT, title TEXT, done INTEGER DEFAULT 0)"
}
dbOps.open(dbName, 1, jsonLib.encode(createSqls), nil)

-- 新增
dbOps.insert(dbName, 'todo', jsonLib.encode({title='写接口文档', done=0}))

-- 查询未完成
local rowsJson = dbOps.query(dbName, 'SELECT * FROM todo WHERE done=?', '[0]')
print(rowsJson)

-- 标记完成
dbOps.update(dbName, 'todo', jsonLib.encode({done=1}), 'id=?', '[1]')

-- 删除已完成
dbOps.delete(dbName, 'todo', 'done=?', '[1]')

dbOps.close(dbName)
```

### 案例 2：配置批量更新（事务）

```lua
import('com.nx.assist.lua.LuaEngine')

local xfxModule = require('lib/XfxPlugin')
local XFX = xfxModule:new({
    apkName = 'xfxPlugin-release.apk',
    context = LuaEngine.getContext(),
})

local dbOps = XFX:getOps('sqliteOps')
local dbName = 'config.db'
local createSqls = {
    "CREATE TABLE IF NOT EXISTS kv(key TEXT PRIMARY KEY, value TEXT)"
}
dbOps.open(dbName, 1, jsonLib.encode(createSqls), nil)

-- 批量更新使用事务
local txSqls = {
    "REPLACE INTO kv(key,value) VALUES('env','prod')",
    "REPLACE INTO kv(key,value) VALUES('feature_x','on')",
    "REPLACE INTO kv(key,value) VALUES('timeout','30')"
}
print('txn result:', dbOps.runInTransaction(dbName, jsonLib.encode(txSqls)))

-- 验证
print(dbOps.query(dbName, "SELECT * FROM kv"))

dbOps.close(dbName)
```

### 案例 3：异步写入日志表

```lua
import('com.nx.assist.lua.LuaEngine')

local xfxModule = require('lib/XfxPlugin')
local XFX = xfxModule:new({
    apkName = 'xfxPlugin-release.apk',
    context = LuaEngine.getContext(),
})

local dbOps = XFX:getOps('sqliteOps')
local dbName = 'log.db'
local createSqls = {
    "CREATE TABLE IF NOT EXISTS logs(id INTEGER PRIMARY KEY AUTOINCREMENT, level TEXT, msg TEXT, ts INTEGER)"
}
dbOps.open(dbName, 1, jsonLib.encode(createSqls), nil)

-- 异步写入
local now = os.time()
dbOps.execSqlAsync(dbName, "INSERT INTO logs(level,msg,ts) VALUES(?,?,?)", jsonLib.encode({'INFO','app started',now}))
sleep(1000)  -- 等待插入完成

-- 稍后查询
print(dbOps.query(dbName, "SELECT * FROM logs ORDER BY id DESC"))

dbOps.close(dbName)
```

### 案例 4：全链路操作（涵盖主要方法）

```lua
import('com.nx.assist.lua.LuaEngine')

local xfxModule = require('lib/XfxPlugin')
local XFX = xfxModule:new({
    apkName = 'xfxPlugin-release.apk',
    context = LuaEngine.getContext(),
})


local dbName = 'demo.db'
local dbOps = XFX:getOps('sqliteOps')

-- 1) 打开并建表
local createSqls = {"CREATE TABLE IF NOT EXISTS demo(id INTEGER PRIMARY KEY AUTOINCREMENT, title TEXT, done INTEGER DEFAULT 0)"}
print('open:', dbOps.open(dbName, 1, jsonLib.encode(createSqls), nil))

-- 2) 插入
print('insert id:', dbOps.insert(dbName, 'demo', jsonLib.encode({title='taskA', done=0})))

-- 3) 查询（无占位符）
print('all:', dbOps.query(dbName, 'SELECT * FROM demo'))

-- 4) 更新（有占位符）
print('update count:', dbOps.update(dbName, 'demo', jsonLib.encode({done=1}), 'id=?', '[1]'))

-- 5) 删除（有占位符）
print('delete count:', dbOps.delete(dbName, 'demo', 'done=?', '[1]'))

-- 6) 事务批量执行
local txSqls = {
    "INSERT INTO demo(title,done) VALUES('taskB',0)",
    "INSERT INTO demo(title,done) VALUES('taskC',0)"
}
print('txn runInTransaction:', dbOps.runInTransaction(dbName, jsonLib.encode(txSqls)))

-- 7) 异步执行（示例写入）
dbOps.execSqlAsync(dbName, "INSERT INTO demo(title,done) VALUES('asyncTask',0)", nil)
sleep(1000)

-- 8) 查询占位符版
print('done=0:', dbOps.query(dbName, 'SELECT * FROM demo WHERE done=?', '[0]'))

-- 9) 清空数据保留表
print('clear:', dbOps.clear(dbName))

-- 10) 关闭并删除数据库文件
dbOps.close(dbName)
print('delete db file:', dbOps.delete(dbName))
```
