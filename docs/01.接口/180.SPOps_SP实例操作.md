## 模块简介

`SPOps` 可通过文件名获取不同实例并进行字符串、整数、布尔的存取，以及键删除与清空。

> SharedPreferences 是Android平台上一个轻量级的存储类，主要用于保存应用的一些常用配置。它通过键值对的形式将数据保存在XML文件中，支持的数据类型包括String、int、long、float和boolean。

> ⚠️ 不支持懒人精灵`最高权限`下运行(最高权限下获取不到Context)。

## 功能概览

- **实例获取**：默认实例、按模式/文件名获取实例
- **读写操作**：String / Int / Long / Boolean 的写入与读取（使用 put/get 重载方法）
- **数据管理**：移除键、清空文件
- **多文件支持**：支持多个 SP 文件

## API 接口列表

### 1. 实例获取

#### getInstance()

获取默认实例（文件名默认为应用包名）。

| 返回值类型 | 说明 |
|-----------|------|
| SPUtils | 默认实例 |

**示例：**
```lua
local spStorge = UtilsMain.spOps()
local sp = spStorge.getInstance()
print(sp)
```

#### getInstance()

指定文件名获取实例。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| spName | String | 是 | SP 文件名 |

| 返回值类型 | 说明 |
|-----------|------|
| SPUtils | 实例 |

**示例：**
```lua
local spStorge = UtilsMain.spOps()
local sp = spStorge.getInstance("demo_sp")
print(sp)
```

### 2. 数据读写操作

#### put()

写入值（重载方法，支持多种类型）。

> 根据 `value` 参数的类型自动选择对应的存储方法。支持 String、Int、Long、Boolean 类型。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| spName | String | 是 | SP 文件名 |
| key | String | 是 | 键 |
| value | String / Int / Long / Boolean | 是 | 值（类型自动识别） |

**重载方法：**
- `put(spName: String, key: String, value: String)` - 写入字符串
- `put(spName: String, key: String, value: Int)` - 写入整数
- `put(spName: String, key: String, value: Long)` - 写入长整数
- `put(spName: String, key: String, value: Boolean)` - 写入布尔值

**示例：**
```lua
local spStorge = UtilsMain.spOps()

-- 写入字符串
spStorge.put("demo_sp", "token", "abc")

-- 写入整数
spStorge.put("demo_sp", "age", 18)

-- 写入长整数
spStorge.put("demo_sp", "ts", 1234567890)

-- 写入布尔值
spStorge.put("demo_sp", "login", true)
```

#### get()

读取值（重载方法，根据默认值类型自动推断返回类型）。

> 根据 `defaultValue` 参数的类型自动选择对应的读取方法。返回类型与默认值类型一致。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| spName | String | 是 | SP 文件名 |
| key | String | 是 | 键 |
| defaultValue | String / Int / Long / Boolean | 是 | 默认值（类型决定返回类型） |

| 返回值类型 | 说明 |
|-----------|------|
| 与 defaultValue 类型相同 | 读取到的值，如果键不存在则返回默认值 |

**重载方法：**
- `get(spName: String, key: String, defaultValue: String): String` - 读取字符串
- `get(spName: String, key: String, defaultValue: Int): Int` - 读取整数
- `get(spName: String, key: String, defaultValue: Long): Long` - 读取长整数
- `get(spName: String, key: String, defaultValue: Boolean): Boolean` - 读取布尔值

**示例：**
```lua
local spStorge = UtilsMain.spOps()

-- 读取字符串
print(spStorge.get("demo_sp", "token", ""))

-- 读取整数
print(spStorge.get("demo_sp", "age", 0))

-- 读取长整数
print(spStorge.get("demo_sp", "ts", 0))

-- 读取布尔值
print(spStorge.get("demo_sp", "login", false))
```

### 5. 数据管理


#### getAll()

读取全部值。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| spName | String | 是 | SP 文件名 |

| 返回值类型 | 说明 |
|-----------|------|
| String | 读取到的全部值，json字符串 |

**示例：**
```lua
local spStorge = UtilsMain.spOps()
print(spStorge.getAll("demo_sp"))
```

#### remove()

移除指定键。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| spName | String | 是 | SP 文件名 |
| key | String | 是 | 键 |

**示例：**
```lua
local spStorge = UtilsMain.spOps()
spStorge.remove("demo_sp", "token")
```

#### clear()

清除指定文件的所有数据。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| spName | String | 是 | SP 文件名 |

**示例：**
```lua
local spStorge = UtilsMain.spOps()
spStorge.clear("demo_sp")
```

## 完整示例

```lua
import('com.nx.assist.lua.LuaEngine')
local loader = LuaEngine.loadApk('xfxPlugin-release.apk'); assert(loader)
local context = LuaEngine.getContext()
local UtilsMain = loader.loadClass('com.xfx.plugin.UtilsMain'); assert(UtilsMain)
UtilsMain.init(context)

local spStorge = UtilsMain.spOps()

local key = "demo_sp"

-- 获取实例
spStorge.getInstance(key)

-- 写入
spStorge.put(key, "token", "abc")
spStorge.put(key, "age", 18)
spStorge.put(key, "login", true)

-- 读取
print("token:", spStorge.get(key, "token", ""))
print("age:", spStorge.get(key, "age", 0))
print("login:", spStorge.get(key, "login", false))
print(spStorge.getAll(key))

-- 删除与清空
spStorge.remove(key, "token")
spStorge.clear(key)
print(spStorge.getAll(key))
```


## OOP 调用例子

```lua
-- ============================================
-- SharedPreferences 操作示例
-- 演示如何使用 XfxPlugin.lua 调用 spStorge 进行数据存储
-- ============================================
import('com.nx.assist.lua.LuaEngine')

print('========== SharedPreferences 操作示例 ==========')

-- 加载 XfxPlugin.lua 类
local xfxModule = require('lib/XfxPlugin')

-- 创建 XFX 对象实例
local XFX = xfxModule:new({
    apkName = 'xfxPlugin-release.apk',  -- APK 文件名
})

XFX:setContext(LuaEngine.getContext())  -- 设置全局上下文

-- 获取 spStorge 对象
local spStorge = XFX:getOps('spOps')
if not spStorge then
    print('无法获取 spStorge 对象')
    return
end

print('开始 SharedPreferences 操作示例...')

-- ============================================
-- 1. 存储数据
-- ============================================
print('\n--- 1. 存储数据 ---')

local spName = 'my_config'

-- 存储字符串
spStorge.put(spName, 'username', 'user123')
print('已存储用户名: user123')

-- 存储整数
spStorge.put(spName, 'level', 10)
print('已存储等级: 10')

-- 存储布尔值
spStorge.put(spName, 'isFirstRun', false)
print('已存储首次运行标志: false')

-- ============================================
-- 2. 读取数据
-- ============================================
print('\n--- 2. 读取数据 ---')

local username = spStorge.get(spName, 'username', '')
print('读取用户名: ' .. username)

local level = spStorge.get(spName, 'level', 0)
print('读取等级: ' .. tostring(level))

local isFirstRun = spStorge.get(spName, 'isFirstRun', true)
print('读取首次运行标志: ' .. tostring(isFirstRun))

-- ============================================
-- 3. 数据管理
-- ============================================
print('\n--- 3. 数据管理 ---')

-- 移除指定键
spStorge.remove(spName, 'username')
print('已移除用户名')

-- 清除所有数据
spStorge.clear(spName)
print('已清除所有数据')

print('\n========== SharedPreferences 操作示例完成 ==========')
```

## 实际应用示例

### 示例 1：用户配置存储

```lua
import('com.nx.assist.lua.LuaEngine')

local xfxModule = require('lib/XfxPlugin')
local XFX = xfxModule:new({apkName = 'xfxPlugin-release.apk'})

XFX:setContext(LuaEngine.getContext())  -- 设置全局上下文

local spStorge = XFX:getOps('spOps')
if not spStorge then
    print('无法获取 spStorge 对象')
    return
end

local configName = 'user_config'

-- 保存用户配置
spStorge.put(configName, 'nickname', '测试用户')
spStorge.put(configName, 'age', 25)
spStorge.put(configName, 'notifications', true)

-- 读取用户配置
local nickname = spStorge.get(configName, 'nickname', '')
local age = spStorge.get(configName, 'age', 0)
local notifications = spStorge.get(configName, 'notifications', false)

print('用户配置:')
print('  昵称: ' .. nickname)
print('  年龄: ' .. tostring(age))
print('  通知: ' .. tostring(notifications))
```


## 注意事项

1.  **初始化要求**：使用前需先调用 `xxxOps.setContext(context)`或`UtilsMain.init(context)` 初始化上下文。
2. **SP 文件名**：
   - SP 文件名用于区分不同的存储文件；同一 `spName` 共享数据；不同文件名数据互不影响。
   - 建议使用有意义的文件名
   - 文件名区分大小写

3. **默认值**：
   - 读取数据时必须提供默认值
   - 如果键不存在，返回默认值
   - 默认值类型必须与读取类型匹配

4. **数据持久化**：
   - 数据会自动持久化到设备
   - 应用卸载后数据会被清除
   - 数据存储在应用私有目录

5. **多文件支持**：
   - 可以使用不同的 SP 文件名存储不同类别的数据
   - 每个文件独立管理
   - 建议按功能模块划分文件
