## 模块简介

`RomOps` 提供常见 ROM 品牌识别与 ROM 信息查询。

## 功能概览

- **品牌判断**：判断 ROM 品牌，共 20 个：华为、Vivo、小米、OPPO、乐视、360、中兴、一加、努比亚、酷派、LG、Google、三星、魅族、联想、锤子、HTC、索尼、金立、摩托罗拉
- **信息获取**：获取 ROM 详情（名称、版本、API Level 等）

## API 接口列表

### 1. 品牌判断

以下接口均返回 Boolean，表示当前 ROM 是否为对应品牌。

| 方法 | 说明 |
|------|------|
| isHuawei() | 是否华为 |
| isVivo() | 是否 Vivo |
| isXiaomi() | 是否小米/MIUI |
| isOppo() | 是否 OPPO |
| isLeeco() | 是否 乐视 |
| is360() | 是否 360 |
| isZte() | 是否 中兴 |
| isOneplus() | 是否 一加 |
| isNubia() | 是否 努比亚 |
| isCoolpad() | 是否 酷派 |
| isLg() | 是否 LG |
| isGoogle() | 是否 Google 原生 |
| isSamsung() | 是否 三星 |
| isMeizu() | 是否 魅族 |
| isLenovo() | 是否 联想 |
| isSmartisan() | 是否 锤子 |
| isHtc() | 是否 HTC |
| isSony() | 是否 索尼 |
| isGionee() | 是否 金立 |
| isMotorola() | 是否 摩托罗拉 |

**示例：**
```lua
local romOps = UtilsMain.romOps()
print("isXiaomi:", romOps.isXiaomi())
print("isHuawei:", romOps.isHuawei())
print("isGoogle:", romOps.isGoogle())
```

### 2. ROM 信息

#### getRomInfo()

获取 ROM 详细信息。

| 返回值类型 | 说明 |
|-----------|------|
| RomInfo | 包含 name/version/apiLevel 等信息 |

**示例：**
```lua
local romOps = UtilsMain.romOps()
local info = romOps.getRomInfo()
print(info)  -- RomInfo.toString() 输出具体信息
```

## 完整示例

```lua
import('com.nx.assist.lua.LuaEngine')
local loader = LuaEngine.loadApk('xfxPlugin-release.apk'); assert(loader)
local context = LuaEngine.getContext()
local UtilsMain = loader.loadClass('com.xfx.plugin.UtilsMain'); assert(UtilsMain)
UtilsMain.init(context)

local romOps = UtilsMain.romOps()

print("Huawei:", romOps.isHuawei())
print("Vivo:", romOps.isVivo())
print("Xiaomi:", romOps.isXiaomi())
print("Oppo:", romOps.isOppo())
print("Leeco:", romOps.isLeeco())
print("360:", romOps.is360())
print("Zte:", romOps.isZte())
print("Oneplus:", romOps.isOneplus())
print("Nubia:", romOps.isNubia())
print("Coolpad:", romOps.isCoolpad())
print("Lg:", romOps.isLg())
print("Google:", romOps.isGoogle())
print("Samsung:", romOps.isSamsung())
print("Meizu:", romOps.isMeizu())
print("Lenovo:", romOps.isLenovo())
print("Smartisan:", romOps.isSmartisan())
print("Htc:", romOps.isHtc())

```

## 注意事项

- **品牌判断覆盖**：接口覆盖主流品牌，未列出的品牌可通过 `getRomInfo()` 查看。
