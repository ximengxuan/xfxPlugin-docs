## 模块简介

`VolumeOps` 提供音量获取/设置、最大最小音量查询，以及振动与取消振动等能力。

> ⚠️ 不支持懒人精灵`最高权限`下运行(最高权限下获取不到Context)。

## 功能概览

- **上下文设置**：设置 Context（setContext，与 UtilsMain.init 二选一）
- **音量查询与设置**：当前音量、最大/最小音量、调节音量
- **振动控制**：启动与取消振动

## API 接口列表

#### setContext()

按需设置 Context（建议传入 applicationContext），在未调用 `UtilsMain.init` 时也可提前注入。

> setContext()和 UtilsMain.init() 二选一调用即可。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| context | Context | 是 | 应用或 Activity 的 context，建议 applicationContext |

| 返回值类型 | 说明 |
|-----------|------|
| void | 无 |

**示例：**
```lua
import('com.nx.assist.lua.LuaEngine')

local volOps = UtilsMain.volumeOps()

local context = LuaEngine.getContext()
volOps.setContext(context)  -- 传入已有的 context
```


### 1. 音量

#### getVolume()

获取指定音频流的当前音量。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| streamType | Int | 是 | 音频流类型，如 `AudioManager.STREAM_MUSIC` |

| 返回值类型 | 说明 |
|-----------|------|
| Int | 当前音量值 |

**示例：**
```lua
import('com.nx.assist.lua.LuaEngine')

local loader = LuaEngine.loadApk('xfxPlugin-release.apk')
local UtilsMain = loader.loadClass('com.xfx.plugin.UtilsMain')

local volOps = UtilsMain.volumeOps()

-- 单独设置上下文
volOps.setContext(LuaEngine.getContext())

local AudioManager = loader.loadClass('android.media.AudioManager')
print('当前音乐音量: ' .. volOps.getVolume(AudioManager.STREAM_MUSIC))
```

#### setVolume()

设置指定音频流的音量。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| streamType | Int | 是 | 音频流类型 |
| volume | Int | 是 | 目标音量值 |
| flags | Int | 是 | 标志位，如 `AudioManager.FLAG_SHOW_UI` |


#### getMaxVolume()

获取指定音频流的最大音量。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| streamType | Int | 是 | 音频流类型 |

| 返回值类型 | 说明 |
|-----------|------|
| Int | 最大音量值 |


#### getMinVolume()

获取指定音频流的最小音量。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| streamType | Int | 是 | 音频流类型 |

| 返回值类型 | 说明 |
|-----------|------|
| Int | 最小音量值 |


### 2. 振动

#### vibrate()

启动振动。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| milliseconds | Long | 是 | 振动时长（毫秒），建议在 100-2000 毫秒之间 |

**示例：**
```lua
local volOps = UtilsMain.volumeOps()
volOps.vibrate(500)
```

注意事项：

- 震动功能需要 `VIBRATE` 权限
- 某些设备可能不支持震动

#### cancel()

取消振动。

**示例：**
```lua
local volOps = UtilsMain.volumeOps()
volOps.cancel()
```

## 完整示例(1)

```lua
import('com.nx.assist.lua.LuaEngine')

local loader = LuaEngine.loadApk('xfxPlugin-release.apk')
local context = LuaEngine.getContext()
local UtilsMain = loader.loadClass('com.xfx.plugin.UtilsMain')

local volOps = UtilsMain.volumeOps()

-- 单独设置上下文
volOps.setContext(context)

local AudioManager = loader.loadClass('android.media.AudioManager')
local STREAM_MUSIC = AudioManager.STREAM_MUSIC
local FLAG_SHOW_UI = AudioManager.FLAG_SHOW_UI

local currentVolume = volOps.getVolume(STREAM_MUSIC)
print('当前音乐音量: ' .. tostring(currentVolume))

local maxVolume = volOps.getMaxVolume(STREAM_MUSIC)
print('最大音量: ' .. tostring(maxVolume))

volOps.setVolume(STREAM_MUSIC, maxVolume / 4, FLAG_SHOW_UI)

-- 震动
volOps.vibrate(5000)
sleep(1000)
volOps.cancel()
sleep(1000)
```

## 完整示例(2)

```lua
import('com.nx.assist.lua.LuaEngine')

local loader = LuaEngine.loadApk('xfxPlugin-release.apk')
local UtilsMain = loader.loadClass('com.xfx.plugin.UtilsMain')

local volOps = UtilsMain.volumeOps()

-- 单独设置上下文
volOps.setContext(LuaEngine.getContext())

local AudioManager = loader.loadClass('android.media.AudioManager')
print('当前音乐音量: ' .. volOps.getVolume(AudioManager.STREAM_MUSIC))
print('当前系统音量: ' .. volOps.getVolume(AudioManager.STREAM_SYSTEM))

-- 设置音量
volOps.setVolume(AudioManager.STREAM_MUSIC, 10, AudioManager.FLAG_SHOW_UI)
sleep(1000)
volOps.setVolume(AudioManager.STREAM_SYSTEM, 5, AudioManager.FLAG_SHOW_UI)

-- 获取音量
print('当前音乐音量: ' .. volOps.getVolume(AudioManager.STREAM_MUSIC))
print('当前系统音量: ' .. volOps.getVolume(AudioManager.STREAM_SYSTEM))

-- 获取最大和最小音量
print('音乐最大音量: ' .. volOps.getMaxVolume(AudioManager.STREAM_MUSIC))
print('音乐最小音量: ' .. volOps.getMinVolume(AudioManager.STREAM_MUSIC))

```

### 常用 AudioManager 常量

```lua
local AudioManager = loader.loadClass('android.media.AudioManager')

-- 音频流类型
local STREAM_MUSIC = AudioManager.STREAM_MUSIC          -- 音乐
local STREAM_RING = AudioManager.STREAM_RING            -- 铃声
local STREAM_VOICE_CALL = AudioManager.STREAM_VOICE_CALL -- 通话
local STREAM_SYSTEM = AudioManager.STREAM_SYSTEM        -- 系统音
local STREAM_ALARM = AudioManager.STREAM_ALARM           -- 闹钟
local STREAM_NOTIFICATION = AudioManager.STREAM_NOTIFICATION -- 通知

-- 标志位
local FLAG_SHOW_UI = AudioManager.FLAG_SHOW_UI          -- 显示音量UI
local FLAG_PLAY_SOUND = AudioManager.FLAG_PLAY_SOUND    -- 播放声音
local FLAG_VIBRATE = AudioManager.FLAG_VIBRATE         -- 震动
```



## 注意事项

- **初始化要求**：使用前需先调用 `xxxOps.setContext(context)`或`UtilsMain.init(context)` 初始化上下文。
- **权限与兼容**：振动需设备支持并可能需要振动权限；音量调整需确保流类型与系统策略允许变更。
