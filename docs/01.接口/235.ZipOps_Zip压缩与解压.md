## 模块简介

`ZipOps` 提供 ZIP 压缩与解压能力，支持单文件压缩、目录压缩，以及带/不带编码参数的解压。

## 功能概览

- **压缩**：文件压缩、目录压缩
- **解压**：解压到指定目录（支持指定编码）

## API 接口列表

### 1. 压缩

#### zipFile()

压缩单个文件。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| srcFilePath | String | 是 | 源文件路径 |
| zipFilePath | String | 是 | 生成的 ZIP 路径 |

| 返回值类型 | 说明 |
|-----------|------|
| Boolean | 成功返回 true |

**示例：**
```lua
local zipOps = UtilsMain.zipOps()
print(zipOps.zipFile("/sdcard/a.txt", "/sdcard/a.zip"))
```

#### zipFiles()

压缩文件夹（递归）。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| srcDirPath | String | 是 | 源目录路径 |
| zipFilePath | String | 是 | 生成的 ZIP 路径 |

| 返回值类型 | 说明 |
|-----------|------|
| Boolean | 成功返回 true |

**示例：**
```lua
local zipOps = UtilsMain.zipOps()
print(zipOps.zipFiles("/sdcard/mydir", "/sdcard/mydir.zip"))
```

### 2. 解压

#### unzipFile()

解压 ZIP 到目标目录。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| zipFilePath | String | 是 | ZIP 文件路径 |
| destDirPath | String | 是 | 目标目录路径 |
| charsetName | String | 否 | 字符编码（如 `UTF-8`） |

| 返回值类型 | 说明 |
|-----------|------|
| Boolean | 成功返回 true |

**示例：**
```lua
local zipOps = UtilsMain.zipOps()
print(zipOps.unzipFile("/sdcard/a.zip", "/sdcard/out"))
print(zipOps.unzipFile("/sdcard/a.zip", "/sdcard/out", "UTF-8"))
```

## 完整示例

```lua
import('com.nx.assist.lua.LuaEngine')
local loader = LuaEngine.loadApk('xfxPlugin-release.apk'); assert(loader)
local context = LuaEngine.getContext()
local UtilsMain = loader.loadClass('com.xfx.plugin.UtilsMain'); assert(UtilsMain)
UtilsMain.init(context)

local zipOps = UtilsMain.zipOps()

print("zip file:", zipOps.zipFile("/sdcard/a.txt", "/sdcard/a.zip"))
print("zip dir:", zipOps.zipFiles("/sdcard/mydir", "/sdcard/mydir.zip"))

print("unzip:", zipOps.unzipFile("/sdcard/a.zip", "/sdcard/out"))
print("unzip charset:", zipOps.unzipFile("/sdcard/a.zip", "/sdcard/out_utf8", "UTF-8"))
```

## 注意事项

- **初始化要求**：使用前需先调用 `UtilsMain.init(context)`。
- **路径权限**：需确保对源文件/目录和目标路径拥有读写权限。
