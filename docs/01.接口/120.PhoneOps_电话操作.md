## 模块简介

`PhoneOps` 提供电话相关操作，包括设备/电话类型查询、SIM 状态与运营商信息、拨号与通话、发送短信等。

## 功能概览

- **设备与类型**：判断是否为电话设备、获取电话类型
- **SIM 信息**：判断 SIM 是否就绪、获取运营商名称与 MNC 运营商
- **通话与短信**：拨号 UI、直接拨打、发送短信、静默发送短信

## API 接口列表

### 1. 设备与类型

#### isPhone()

判断当前设备是否为电话设备。

| 返回值类型 | 说明 |
|-----------|------|
| Boolean | 是电话返回 true |

**示例：**
```lua
local phOps = UtilsMain.phoneOps()
print(phOps.isPhone())
```


#### getPhoneType()

获取电话类型。

| 返回值类型 | 说明 |
|-----------|------|
| Int | 电话类型常量 |

**示例：**
```lua
local phOps = UtilsMain.phoneOps()
print(phOps.getPhoneType())
```

### 2. SIM 信息

#### isSimCardReady()

判断 SIM 卡是否已就绪。

| 返回值类型 | 说明 |
|-----------|------|
| Boolean | 就绪返回 true |

**示例：**
```lua
local phOps = UtilsMain.phoneOps()
print(phOps.isSimCardReady())
```

#### getSimOperatorName()

获取 SIM 运营商名称。

| 返回值类型 | 说明 |
|-----------|------|
| String | 运营商名称 |

**示例：**
```lua
local phOps = UtilsMain.phoneOps()
print(phOps.getSimOperatorName())
```

#### getSimOperatorByMnc()

根据 MNC 获取运营商标识。

| 返回值类型 | 说明 |
|-----------|------|
| String | 运营商标识 |

**示例：**
```lua
local phOps = UtilsMain.phoneOps()
print(phOps.getSimOperatorByMnc())
```

### 3. 通话与短信

#### dial()

打开拨号界面（无需 `CALL_PHONE` 权限）。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| number | String | 是 | 目标号码 |

| 返回值类型 | 说明 |
|-----------|------|
| void | 无 |

**示例：**
```lua
local phOps = UtilsMain.phoneOps()
phOps.dial("10086")
```

#### call()

直接拨打电话（需要 `CALL_PHONE` 权限）。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| number | String | 是 | 目标号码 |

| 返回值类型 | 说明 |
|-----------|------|
| void | 无 |

**示例：**
```lua
local phOps = UtilsMain.phoneOps()
phOps.call("10086")
```

#### sendSms()

调用默认短信程序发送短信。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| number | String | 是 | 目标号码 |
| content | String | 是 | 短信内容 |

| 返回值类型 | 说明 |
|-----------|------|
| void | 无 |

**示例：**
```lua
local phOps = UtilsMain.phoneOps()
phOps.sendSms("10086", "hello")
```

#### sendSmsSilent()

静默发送短信。必须持有 `SEND_SMS` 权限。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| phoneNumber | String? | 是 | 目标号码 |
| content | String? | 是 | 短信内容 |

| 返回值类型 | 说明 |
|-----------|------|
| void | 无返回值 |

**示例：**
```lua
local phOps = UtilsMain.phoneOps()
-- 静默发送短信（需要 SEND_SMS 权限）
phOps.sendSmsSilent("10086", "hello")
```

## 完整示例

```lua
import('com.nx.assist.lua.LuaEngine')
local loader = LuaEngine.loadApk('xfxPlugin-release.apk'); assert(loader)
local context = LuaEngine.getContext()
local UtilsMain = loader.loadClass('com.xfx.plugin.UtilsMain'); assert(UtilsMain)
UtilsMain.init(context)

local phOps = UtilsMain.phoneOps()

print("isPhone:", phOps.isPhone())
print("deviceId:", phOps.getDeviceId())
print("phoneType:", phOps.getPhoneType())
print("simReady:", phOps.isSimCardReady())
print("simOperator:", phOps.getSimOperatorName())
print("simMnc:", phOps.getSimOperatorByMnc())

phOps.dial("10086")
phOps.call("10086")
phOps.sendSms("10086", "hello")
-- 静默发送短信（需要 SEND_SMS 权限）
phOps.sendSmsSilent("10086", "hello")
```

## 注意事项

- **初始化要求**：使用前需先调用 `UtilsMain.init(context)`。
- **权限**：
  - `call()` 需要 `CALL_PHONE` 权限
  - `sendSms()` 调用系统短信应用，通常无需短信发送权限
  - `sendSmsSilent()` 必须持有 `SEND_SMS` 权限
