## 模块简介

`FfmpegOps` 封装 FFmpeg 常用视频处理能力，含同步与异步接口（通过线程池执行），支持帧提取、信息获取、格式转换、裁剪/合并、提取音频、执行自定义命令，以及一系列画面调整滤镜。可设置回调用于向上层通知进度/状态。

## 功能概览

- **回调与命令执行**：设置回调、同步/异步执行命令，检查可用性、获取版本
- **基础处理**：提取图片帧、获取视频信息、宽高、帧率调整、翻转
- **格式与片段**：格式转换、裁剪（含边距裁剪）、合并、截取音频
- **画面处理滤镜**：缩放、裁剪画面、色彩/色调/RGB/伽马、淡入淡出、暗角、降噪、锐化、高斯模糊

> 说明：以下“同步/异步”合并描述，异步版名称后缀 `Async`，行为一致但在后台线程执行。

## API 接口列表

### 1. 回调与命令执行

#### setMessageCallback()

设置消息回调（在主线程执行）。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| callback | Runnable? | 否 | 回调，为 null 则清除 |

| 返回值类型 | 说明 |
|-----------|------|
| void | 无 |

**示例：**
```lua
local ffOps = UtilsMain.ffmpegOps()
ffOps.setMessageCallback(function() print("ffmpeg callback") end)
```

#### executeCommand() 
#### executeCommandAsync()

同步/异步执行自定义 FFmpeg 命令，可选进度回调。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| command | String | 是 | FFmpeg 命令行（完整参数串） |
| operationName | String | 否 | 操作名称，用于回调提示，默认 "FFMPEG操作" |
| progressCallback | ((Statistics) -> Unit)? | 否 | 进度回调（仅同步方法支持；异步版无此参数） |

| 返回值类型 | 说明 |
|-----------|------|
| Boolean | 成功返回 true |

**示例：**
```lua
local ffOps = UtilsMain.ffmpegOps()
local ok = ffOps.executeCommand("-version", "检查版本")
print(ok)
ffOps.executeCommandAsync("-i /sdcard/in.mp4 -vn -acodec copy -y /sdcard/out.aac", "抽音频")
```

#### isEnable()

检查 FFmpeg 是否可用。

| 返回值类型 | 说明 |
|-----------|------|
| Boolean | 可用返回 true |

**示例：**
```lua
local ffOps = UtilsMain.ffmpegOps()
print(ffOps.isEnable())
```

#### getVersion()

获取 FFmpeg 版本信息字符串。

| 返回值类型 | 说明 |
|-----------|------|
| String | 版本信息或错误提示 |

**示例：**
```lua
local ffOps = UtilsMain.ffmpegOps()
print(ffOps.getVersion())
```

### 2. 帧提取与信息

#### extractFrame() 
#### extractFrameAsync()

提取指定时间帧（默认首帧）到图片。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| inputVideoPath | String | 是 | 输入视频路径 |
| outputImagePath | String | 是 | 输出图片路径 |
| timeOffset | Double | 是 | 帧时间（秒），0 表示首帧 |

| 返回值类型 | 说明 |
|-----------|------|
| Boolean | 成功返回 true（异步无返回） |

**示例：**
```lua
local ffOps = UtilsMain.ffmpegOps()
ffOps.extractFrame("/sdcard/in.mp4", "/sdcard/cover.jpg", 1.5)
```

#### getInfo()

获取视频信息，返回 JSON 字符串（含 duration、width、height、codec、fps、bitrate 等）。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| videoPath | String | 是 | 视频文件路径 |

| 返回值类型 | 说明 |
|-----------|------|
| String | JSON 字符串 |

**示例：**
```lua
local ffOps = UtilsMain.ffmpegOps()
print(ffOps.getInfo("/sdcard/in.mp4"))
-- {"duration":12,"durationFormatted":"00:00:12.32","codec":"h264 (avc1 \/ 0x31637661)","width":1920,"height":1080,"fps":29.56,"bitrate":20144,"filePath":"\/sdcard\/test\/test.mp4","fileSize":31012174}
```

#### getWidth()

获取视频宽度，失败返回 -1。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| videoPath | String | 是 | 视频文件路径 |

| 返回值类型 | 说明 |
|-----------|------|
| Int | 视频宽度（像素），失败返回 -1 |

**示例：**
```lua
local ffOps = UtilsMain.ffmpegOps()
print(ffOps.getWidth("/sdcard/in.mp4"))
```

#### getHeight()

获取视频高度，失败返回 -1。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| videoPath | String | 是 | 视频文件路径 |

| 返回值类型 | 说明 |
|-----------|------|


### 3. 格式转换与片段裁剪

裁剪时间片段（起始+可选时长）。

#### merge() 
#### mergeAsync()

合并多个视频（内部生成 concat 列表）。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|

|-----------|------|

**示例：**

## 完整示例

同步操作带回调例子：

```lua
import('com.nx.assist.lua.LuaEngine')
import('java.lang.Runnable')

local loader = LuaEngine.loadApk('xfxPlugin-release.apk'); assert(loader)
local UtilsMain = loader.loadClass('com.xfx.plugin.UtilsMain'); assert(UtilsMain)
local ffOps = UtilsMain.ffmpegOps()

-- 设置简单回调
ffOps.setMessageCallback(Runnable(
    {
        run = function()
            print("ffmpeg done")
        end
    }
))

print("=== 视频信息 ===")
print(ffOps.getInfo("/sdcard/test/test.mp4"))

print("\\n=== 提取帧 ===")
ffOps.extractFrame("/sdcard/test/test.mp4", "/sdcard/test/frame.jpg", 0.5)

print("\\n=== 格式转换 ===")
ffOps.convertFormat("/sdcard/test/test.mp4", "/sdcard/test/out.mov", "mov")

print("\\n=== 裁剪与合并 ===")
ffOps.cut("/sdcard/test/test.mp4", "/sdcard/test/cut.mp4", 2.0, 5.0)
ffOps.merge(jsonLib.encode({"/sdcard/test/cut.mp4", "/sdcard/test/test.mp4"}), "/sdcard/test/merge.mp4")

-- print("\\n=== 抽音频 ===")
-- 提取音频（暂不支持）
-- ffOps.extractAudio("/sdcard/test/test.mp4", "/sdcard/test/out.aac", "aac")

print("\\n=== 画面处理 ===")
-- 调整尺寸
ffOps.resize("/sdcard/test/test.mp4", "/sdcard/test/resized.mp4", 720, 1280, true)
print('缩放后的视频信息：',ffOps.getInfo("/sdcard/test/resized.mp4"))
-- -- 色彩均衡（暂不支持）
-- ffOps.adjustColorBalance("/sdcard/test/test.mp4", "/sdcard/test/color.mp4", 0.05, 0.1, 0.15)
-- 淡入淡出
ffOps.addFadeEffect("/sdcard/test/test.mp4", "/sdcard/test/fade.mp4", 1.0, 1.0)
-- 锐化
ffOps.sharpen("/sdcard/test/test.mp4", "/sdcard/test/sharp.mp4", 1.0)

sleep(30000)
```


## OOP调用例子

```lua
-- ============================================
-- FFMPEG 视频处理示例
-- 演示如何使用 XfxPlugin.lua 调用 FfmpegOps 进行视频处理
-- ============================================

-- 使用 XfxPlugin.lua 面向对象封装（推荐）
-- ============================================
print('========== 使用 XfxPlugin.lua 封装 ==========')

-- 加载 XfxPlugin.lua 类
local xfxModule = require('lib/XfxPlugin')

-- 创建 XFX 对象实例
local XFX = xfxModule:new({
    apkName = 'xfxPlugin-release.apk',  -- APK 文件名
})

-- 获取 ffmpegOps 对象
local ffmpegOps = XFX:getOps('ffmpegOps')
if not ffmpegOps then
    print('无法获取 ffmpegOps 对象')
    return
end

-- 检查 FFMPEG 是否可用
local isAvailable = ffmpegOps.isEnable()
print('FFMPEG 是否可用: ' .. tostring(isAvailable))

if not isAvailable then
    print('FFMPEG 不可用，无法继续')
    return
end

-- 获取 FFMPEG 版本信息
```

## 注意事项

- **路径与权限**：确保读写存储权限与路径可访问；输入输出文件不可相同。确保输入视频文件存在且路径正确
- **性能与阻塞**：同步方法会阻塞调用线程；耗时操作建议使用异步接口。
- **参数有效性**：帧率、尺寸、裁剪/时长等参数需合法；滤镜强度过大可能导致画质异常。
- **输出路径**：确保输出目录有写入权限
- **文件格式**：输出图片格式由文件扩展名决定（.jpg, .png 等）
- **异步操作**：异步方法不会阻塞脚本执行，适合长时间操作
- **错误处理**：建议在实际使用中添加错误处理和文件存在性检查
- **命令安全**：`executeCommand` 直接执行 FFmpeg 命令，请确保命令安全、转义妥当。
