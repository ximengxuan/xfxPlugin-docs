## 模块简介

`StringOps` 是字符串操作工具模块，封装了常用的字符串处理功能，包括字符串验证、转换、格式化、JSON 处理、相似度计算等。

## 功能概览

- **字符串验证**：判断是否为空/空白、是否包含中文、是否为纯数字、提取数字、身份证成年校验、是否为普通字面值字符串
- **字符串比较**：区分大小写/忽略大小写的相等判断（equals、equalsIgnoreCase）
- **字符串转换**：首字母大小写转换、反转、全角半角互转、汉字转拼音（全拼/首字母）、null 转空串
- **字符串处理**：获取长度、去除空格与标点符号、按索引取分割项
- **URL 提取**：从文本中提取所有网址（JSON 数组）
- **JSON 处理**：字符串拆分、JSON 数组/对象内字符串数字转整数、去重、随机选择、判断是否包含
- **数组操作**：原地反转字符串数组（reverseStringArrayInPlace）
- **相似度计算**：Jaro-Winkler 相似度算法、汉明距离

## API 接口列表

### 1. 字符串验证类

#### isEmpty()

判断字符串是否为空或长度为 0。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| text | String? | 是 | 输入文本，可为 null |

| 返回值类型 | 说明 |
|-----------|------|
| Boolean | 为空返回 true，否则返回 false |

**示例：**
```lua
local strOps = UtilsMain.stringOps()
print(strOps.isEmpty(""))      -- 输出: true
print(strOps.isEmpty("hello")) -- 输出: false
print(strOps.isEmpty(nil))     -- 输出: true
```

- 18 位：按校验码精确校验
- 15 位：按格式校验并解析出生日期

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| idCard | String? | 是 | 身份证号码，可为 null |

| 返回值类型 | 说明 |
|-----------|------|
| Boolean | 满 18 周岁返回 true；号码不合法、解析失败或未成年返回 false |

**示例：**
```lua

print(strOps.isAdultByIdCard("110105201201010021")) -- 输出: false
```

### 8. 相似度计算

#### jaroWinklerSim()

计算两个字符串的 Jaro-Winkler 相似度。

推荐用于等长字符串的比较。

Jaro-Winkler 相似度是 Jaro 相似度的改进版本，对前缀匹配给予更高权重。返回值范围：0.0（完全不同）到 1.0（完全相同）。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| s1 | String? | 是 | 第一个字符串 |
| s2 | String? | 是 | 第二个字符串 |

| 返回值类型 | 说明 |
|-----------|------|
| Double | 相似度值（0.0 ~ 1.0），若任一输入为 null 则返回 0.0 |

**示例：**
```lua
local strOps = UtilsMain.stringOps()
local similarity1 = strOps.jaroWinklerSim("hello", "hello")
print(similarity1)  -- 输出: 1.0

local similarity2 = strOps.jaroWinklerSim("hello", "hallo")
print(similarity2)  -- 输出: 约 0.88

local similarity3 = strOps.jaroWinklerSim("hello", "world")
print(similarity3)  -- 输出: 约 0.47
```

#### hmDistance()

计算两个等长字符串的汉明距离（不同字符位置的个数）。

> 汉明距离表示两个等长字符串对应位置上不同字符的数量。例如 `"110011"` 与 `"100010"` 在第3位和第5位不同，汉明距离为 2。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| s1 | String? | 是 | 第一个字符串 |
| s2 | String? | 是 | 第二个字符串 |

| 返回值类型 | 说明 |
|-----------|------|
| Int | 汉明距离（不同字符位置的个数）；若任一输入为 null、或长度不相等，返回 -1 |

**示例：**
```lua
local strOps = UtilsMain.stringOps()

-- 等长字符串比较
local dist1 = strOps.hmDistance("110011", "100010")
print(dist1)  -- 输出: 2（第3位和第5位不同）

-- 完全相同
local dist2 = strOps.hmDistance("hello", "hello")
print(dist2)  -- 输出: 0

-- 长度不等
local dist3 = strOps.hmDistance("abc", "abcd")
print(dist3)  -- 输出: -1

-- null 输入
local dist4 = strOps.hmDistance("abc", nil)
print(dist4)  -- 输出: -1
```

```lua
    local similarity = strOps.jaroWinklerSim(keyword, candidate)

        })
    end

    return a.similarity > b.similarity
end)
```

## 注意事项

- **初始化要求**：使用前必须先调用 `UtilsMain.init(context)` 完成插件初始化。

- **null 处理**：大部分方法都支持 null 输入，会返回相应的默认值（如空字符串、false、0 等）。

- **JSON 格式**：JSON 相关方法返回的都是 JSON 字符串，在 Lua 中需要使用 `jsonLib.decode()` 解析

- **参数完整性**：调用插件接口时，所有方法必须提供完整的参数，参数数量和类型必须与插件的接口完全一致。

- **错误处理**：JSON 处理方法在遇到非法输入时会返回默认值（如 `"[]"` 或 `"{}"`），不会抛出异常。
