## 模块简介

[MMKV](https://github.com/Tencent/MMKV) 是基于 mmap 内存映射的高性能键值存储框架。从 2015 年中至今在微信上使用，其性能和稳定性经过了时间的验证。专为替代安卓原生的 SharedPreferences（SP）设计，解决了 SP 性能差、ANR、数据丢失等问题。支持多进程访问，数据存储更安全可靠。



> ⚠️ 不支持懒人精灵`最高权限`下运行(最高权限下获取不到Context)。

## 功能概览

- **实例配置**：setInstance 设置默认实例 ID，用于隔离不同数据
- **高性能存储**：基于 mmap 内存映射，读写性能极高
- **多进程支持**：支持多进程安全访问
- **数据类型支持**：String / Int / Long / Boolean / Float / Double（put/get 重载）
- **数据管理**：put 写入、get 读取、remove 移除键、clear 清空、isExists 检查键是否存在、getAllKeys 获取所有键名
- **存储统计**：totalSize 获取存储总大小、actualSize 获取实际使用大小

## API 接口列表

### 1. 默认实例配置

#### setInstance()

设置默认 MMKV 实例 ID（创建或获取指定 ID 的实例）。

> 不调用这个接口时，使用默认实例；如需隔离数据，可设置专用实例 ID。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| instanceId | String | 是 | MMKV 实例 ID |

**示例：**
```lua
local MMKV = UtilsMain.mmkvOps()
MMKV.setInstance("my_app_config")
```

### 2. 数据读写操作

#### put()

写入值（重载方法，支持多种类型）。

> 根据 `value` 参数的类型自动选择对应的存储方法。支持 String、Int、Long、Boolean、Float、Double 类型。

⚠️ 写值的时候必须和之前写入的值的数据类型相同。否则可能会出现报错。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| key | String | 是 | 键 |
| value | String / Int / Long / Boolean / Float / Double | 是 | 值（类型自动识别） |

| 返回值类型 | 说明 |
|-----------|------|
| Boolean | 是否成功 |

**重载方法：**
- `put(key: String, value: String)` - 写入字符串
- `put(key: String, value: Int)` - 写入整数
- `put(key: String, value: Long)` - 写入长整数
- `put(key: String, value: Boolean)` - 写入布尔值
- `put(key: String, value: Float)` - 写入浮点数
- `put(key: String, value: Double)` - 写入双精度浮点数

**示例：**
```lua
local MMKV = UtilsMain.mmkvOps()

-- 写入字符串
MMKV.put("token", "abc123")

-- 写入整数
MMKV.put("age", 18)

-- 写入长整数
MMKV.put("ts", 1234567890)

-- 写入布尔值
MMKV.put("login", true)

-- 写入浮点数
MMKV.put("score", 95.5)

-- 写入双精度浮点数
MMKV.put("price", 99.99)
```

**注意：**

如果之前已经写入了数据，那么再次写入数据特别是变量获取到的数据的时候，数据类型必须和原有的类型一致。否则会出现报错。
例如，原来写入了整数型，后面又要写入小数型，就会出现无法写入导致脚本报错。

为了避免这类问题，建议在写值的时候，使用异常捕获避免脚本挂掉。

```lua
xpcall(function()
    MMKV.put('level', level)   -- 保存到本地二进制文件中
end, function(e)
    print(e)
end)
```

#### get()

读取值（重载方法，根据默认值类型自动推断返回类型）。

> 根据 `defaultValue` 参数的类型自动选择对应的读取方法。返回类型与默认值类型一致。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| key | String | 是 | 键 |
| defaultValue | String / Int / Long / Boolean / Float / Double | 是 | 默认值（类型决定返回类型） |

| 返回值类型 | 说明 |
|-----------|------|
| 与 defaultValue 类型相同 | 读取到的值，如果键不存在则返回默认值 |

**重载方法：**
- `get(key: String, defaultValue: String): String` - 读取字符串
- `get(key: String, defaultValue: Int): Int` - 读取整数
- `get(key: String, defaultValue: Long): Long` - 读取长整数（defaultValue 可选，默认为 0）
- `get(key: String, defaultValue: Boolean): Boolean` - 读取布尔值
- `get(key: String, defaultValue: Float): Float` - 读取浮点数
- `get(key: String, defaultValue: Double): Double` - 读取双精度浮点数

**示例：**
```lua
local MMKV = UtilsMain.mmkvOps()

-- 读取字符串
print(MMKV.get("token", ""))

-- 读取整数
print(MMKV.get("age", 0))

-- 读取长整数
print(MMKV.get("ts", 0))

-- 读取布尔值
print(MMKV.get("login", false))

-- 读取浮点数
print(MMKV.get("score", 0.0))

-- 读取双精度浮点数
print(MMKV.get("price", 0.0))
```

### 3. 数据管理

#### remove()

移除指定键或多个键（自动检测参数类型）。

> 如果参数是单个键（字符串），则移除该键；如果参数是 JSON 数组格式，则批量移除多个键。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| keyOrKeys | String | 是 | 单个键名，或 JSON 数组字符串（如：`["key1", "key2"]`） |

| 返回值类型 | 说明 |
|-----------|------|
| Boolean | 是否成功 |

**示例：**
```lua
local MMKV = UtilsMain.mmkvOps()

-- 移除单个键
MMKV.remove("token")

-- 移除多个键（JSON 数组格式）
MMKV.remove('["key1", "key2"]')

-- 或使用 jsonLib
MMKV.remove(jsonLib.encode({"key1", "key2"}))
```

#### clear()

清除所有数据。

| 返回值类型 | 说明 |
|-----------|------|
| Boolean | 是否成功 |

**示例：**
```lua
local MMKV = UtilsMain.mmkvOps()
MMKV.clear()
```

#### isExists()

检查键是否存在。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| key | String | 是 | 键 |

| 返回值类型 | 说明 |
|-----------|------|
| Boolean | 是否存在 |

**示例：**
```lua
local MMKV = UtilsMain.mmkvOps()
if MMKV.isExists("token") then
    print("token 存在")
end
```

### 4. 存储统计

#### totalSize()

获取存储的总大小（字节数）。

| 返回值类型 | 说明 |
|-----------|------|
| Long | 总大小（字节），失败返回 -1 |

**示例：**
```lua
local MMKV = UtilsMain.mmkvOps()
print("总大小:", MMKV.totalSize(), "字节")
```

#### actualSize()

获取实际使用的存储大小（字节数）。

| 返回值类型 | 说明 |
|-----------|------|
| Long | 实际使用大小（字节），失败返回 -1 |

**示例：**
```lua
local MMKV = UtilsMain.mmkvOps()
print("实际使用:", MMKV.actualSize(), "字节")
```

## 完整示例

```lua
import('com.nx.assist.lua.LuaEngine')
local loader = LuaEngine.loadApk('xfxPlugin-release.apk'); assert(loader)
local context = LuaEngine.getContext()
local UtilsMain = loader.loadClass('com.xfx.plugin.UtilsMain'); assert(UtilsMain)
UtilsMain.init(context)

local MMKV = UtilsMain.mmkvOps()

-- 设置默认实例 ID（可选）
MMKV.setInstance("my_app_config")

-- 存储数据
MMKV.put("token", "abc123")
MMKV.put("age", 18)
MMKV.put("ts", 1234567890)
MMKV.put("login", true)
MMKV.put("score", 95.5)
MMKV.put("price", 99.99)

-- 读取数据
print("token:", MMKV.get("token", ""))
print("age:", MMKV.get("age", 0))
print("ts:", MMKV.get("ts", 0))
print("login:", MMKV.get("login", false))
print("score:", MMKV.get("score", 0.0))
print("price:", MMKV.get("price", 0.0))

-- 检查键是否存在
if MMKV.isExists("token") then
    print("token 存在")
end

-- 获取所有数据
print("所有键:", MMKV.getAllKeys())

-- 获取存储统计
print("总大小:", MMKV.totalSize(), "字节")
print("实际使用:", MMKV.actualSize(), "字节")

-- 移除键
MMKV.remove("token")

-- 移除多个键（JSON 数组格式）
MMKV.remove('["age", "score"]')
-- 或使用 jsonLib
-- MMKV.remove(jsonLib.encode({"age", "score"}))
print("所有键:", MMKV.getAllKeys())

-- 清空所有数据
MMKV.clear()
print("清空后:", MMKV.getAllKeys())
```

## OOP 调用例子

```lua
-- ============================================
-- MMKV 存储操作示例
-- 演示如何使用 XfxPlugin.lua 调用 MMKV 进行高性能数据存储
-- ============================================
import('com.nx.assist.lua.LuaEngine')

print('========== MMKV 存储操作示例 ==========')

-- 加载 XfxPlugin.lua 类
local xfxModule = require('lib/XfxPlugin')

-- 创建 XFX 对象实例
local XFX = xfxModule:new({
    apkName = 'xfxPlugin-release.apk',  -- APK 文件名
    context = LuaEngine.getContext(),  -- 上下文对象
})

-- 获取 MMKV 对象
local MMKV = XFX:getOps('mmkvOps')
if not MMKV then
    print('无法获取 MMKV 对象')
    return
end

print('开始 MMKV 存储操作示例...')

-- ============================================
-- 1. 设置默认实例 ID（可选）
-- ============================================
print('\n--- 1. 设置默认实例 ID ---')

MMKV.setInstance('my_app_config')
print('已设置默认实例 ID: my_app_config')

-- ============================================
-- 2. 存储数据
-- ============================================
print('\n--- 2. 存储数据 ---')

-- 存储字符串
MMKV.put('user_name', 'feiyun')
print('已存储用户名: feiyun')

-- 存储整数
MMKV.put('login_count', 3)
print('已存储登录次数: 3')

-- 存储长整数
local timestamp = os.time() * 1000
MMKV.put('last_login_time', timestamp)
print('已存储最后登录时间: ' .. tostring(timestamp))

-- 存储布尔值
MMKV.put('is_vip', true)
print('已存储 VIP 标志: true')

-- 存储浮点数
MMKV.put('user_score', 95.5)
print('已存储用户分数: 95.5')

-- 存储双精度浮点数
MMKV.put('account_balance', 999.99)
print('已存储账户余额: 999.99')

-- ============================================
-- 3. 读取数据
-- ============================================
print('\n--- 3. 读取数据 ---')

local name = MMKV.get('user_name', 'guest')
print('读取用户名: ' .. name)

local count = MMKV.get('login_count', 0)
print('读取登录次数: ' .. tostring(count))

local lastTime = MMKV.get('last_login_time', 0)
print('读取最后登录时间: ' .. tostring(lastTime))

local isVip = MMKV.get('is_vip', false)
print('读取 VIP 标志: ' .. tostring(isVip))

local score = MMKV.get('user_score', 0.0)
print('读取用户分数: ' .. tostring(score))

local balance = MMKV.get('account_balance', 0.0)
print('读取账户余额: ' .. tostring(balance))

-- ============================================
-- 4. 数据管理
-- ============================================
print('\n--- 4. 数据管理 ---')

-- 检查键是否存在
if MMKV.isExists('user_name') then
    print('user_name 键存在')
end

-- 获取所有键
local allKeys = MMKV.getAllKeys()
print('所有键: ' .. allKeys)

-- 获取存储统计
local totalSize = MMKV.totalSize()
local actualSize = MMKV.actualSize()
print('存储总大小: ' .. tostring(totalSize) .. ' 字节')
print('实际使用大小: ' .. tostring(actualSize) .. ' 字节')

-- 移除指定键
MMKV.remove('user_name')
print('已移除用户名')

-- 移除多个键（JSON 数组格式）
MMKV.remove('["login_count", "user_score"]')
-- 或使用 jsonLib
-- MMKV.remove(jsonLib.encode({"login_count", "user_score"}))
print('已移除多个键')

allKeys = MMKV.getAllKeys()
print('所有键: ' .. allKeys)

-- 清除所有数据
MMKV.clear()
print('已清除所有数据')

print('\n========== MMKV 存储操作示例完成 ==========')
```

## 注意事项

1. **初始化要求**：使用前需先调用 `UtilsMain.init(context)` 初始化上下文，MMKV 会在初始化时自动设置。

2. **默认实例**：
   - 使用前可以设置默认实例 ID 来隔离数据
   - 如果不设置，使用系统默认实例
   - 所有操作都使用同一个默认实例

3. **与 SharedPreferences 的区别**：

| 特性                | MMKV                          | SharedPreferences（对比）|
|---------------------|-------------------------------|-----------------------------|
| 性能                | 读写速度提升 10~100 倍        | 频繁读写易卡顿、ANR         |
| 数据格式            | 二进制（protobuf）| XML 明文（体积大、解析慢）|
| 多进程支持          | 原生支持（无需额外处理）| 需 MODE_MULTI_PROCESS（不可靠） |
| 数据安全性          | 可选加密（AES-128/256）| 明文存储（无加密）|
| 兼容性              | 支持 Android 4.0+（API 14+）| 支持 Android 1.0+           |
