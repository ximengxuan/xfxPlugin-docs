## 模块简介

`ThreadOps` 提供线程相关便捷接口，包括主线程/后台执行、延迟执行与主线程判断。

## 功能概览

- **主线程执行**：立即或延迟在主线程运行任务
- **后台执行**：IO 线程执行任务
- **线程判断**：判断当前是否主线程

## API 接口列表

### runOnUiThread()

在主线程执行任务。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| runnable | Runnable | 是 | 要执行的任务 |

**示例：**
```lua
local threadOps = UtilsMain.threadOps()
threadOps.runOnUiThread(Runnable{
    run = function()
        print("UI thread task")
    end
})
```

### runOnBackground()

在后台（IO 线程）执行任务。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| runnable | Runnable | 是 | 要执行的任务 |

**示例：**
```lua
local threadOps = UtilsMain.threadOps()
threadOps.runOnBackground(Runnable{
    run = function()
        print("background task")
    end
})
```

### runOnUiThreadDelayed()

在主线程延迟执行任务。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| runnable | Runnable | 是 | 要执行的任务 |
| delayMillis | Long | 是 | 延迟毫秒数 |

**示例：**
```lua
local threadOps = UtilsMain.threadOps()
threadOps.runOnUiThreadDelayed(Runnable{
    run = function()
        print("delayed task")
    end
}, 1000)
```

### isMainThread()

判断当前线程是否主线程。

| 返回值类型 | 说明 |
|-----------|------|
| Boolean | 主线程返回 true |

**示例：**
```lua
local threadOps = UtilsMain.threadOps()
print(threadOps.isMainThread())
```

## 完整示例

```lua
import('com.nx.assist.lua.LuaEngine')
import('java.lang.Runnable')

local loader = LuaEngine.loadApk('xfxPlugin-release.apk'); assert(loader)
local context = LuaEngine.getContext()
local UtilsMain = loader.loadClass('com.xfx.plugin.UtilsMain'); assert(UtilsMain)
UtilsMain.init(context)

local threadOps = UtilsMain.threadOps()

threadOps.runOnUiThread(Runnable{
    run = function()
        print("UI task")
    end
})

threadOps.runOnBackground(Runnable{
    run = function()
        print("BG task")
    end
})

threadOps.runOnUiThreadDelayed(Runnable{
    run = function()
        print("Delayed task")
    end
}, 500)

print("isMain:", threadOps.isMainThread())

sleep(2000) -- 等待2秒，以便观察UI线程和后台线程的执行结果
```
