## 模块简介

`ShellOps` 提供 Shell 命令执行能力，支持返回执行输出或仅判断成功与否。

## 功能概览

- **同步执行**：执行命令并获取输出
- **成功判断**：执行命令并返回是否成功

## API 接口列表

### execCmd()

执行命令（同步），返回标准输出。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| command | String | 是 | 命令字符串 |
| isRoot | Boolean | 是 | 是否以 root 执行 |

| 返回值类型 | 说明 |
|-----------|------|
| String | 成功输出内容；失败为空串 |

**示例：**
```lua
local shellOps = UtilsMain.shellOps()
local out = shellOps.execCmd("ls /sdcard", false)
print(out)
```

### execCmdSuccess()

执行命令并返回是否成功。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| command | String | 是 | 命令字符串 |
| isRoot | Boolean | 是 | 是否以 root 执行 |

| 返回值类型 | 说明 |
|-----------|------|
| Boolean | 成功返回 true |

**示例：**
```lua
local shellOps = UtilsMain.shellOps()
print(shellOps.execCmdSuccess("mkdir -p /sdcard/tmp", false))
```

## 完整示例

```lua
import('com.nx.assist.lua.LuaEngine')
local loader = LuaEngine.loadApk('xfxPlugin-release.apk'); assert(loader)
local context = LuaEngine.getContext()
local UtilsMain = loader.loadClass('com.xfx.plugin.UtilsMain'); assert(UtilsMain)
UtilsMain.init(context)

local shellOps = UtilsMain.shellOps()

print("exec out:", shellOps.execCmd("ls /sdcard", false))
print("exec ok:", shellOps.execCmdSuccess("mkdir -p /sdcard/tmp", false))
```

## 注意事项

- **root 权限**：当 `isRoot` 为 true 时需设备已 root；否则命令可能失败。
