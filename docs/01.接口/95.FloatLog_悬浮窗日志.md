## 模块简介

`FloatLog` 提供悬浮日志窗口：创建/显示/移动/缩放/最小化、日志添加导出、计时器与进度显示、Toast 弹窗等。

> ⚠️ 不支持懒人精灵`最高权限`下运行(最高权限下获取不到Context)。

## 功能概览

- **上下文设置**：设置 Context（setContext，与 UtilsMain.init 二选一）
- **窗口管理**：显示、移动、缩放、最小化/恢复、完全显示/隐藏、透明度、圆角、状态导出/关闭
- **日志操作**：添加日志（多重样式）、清空、获取 JSON、导出文件、最大条数限制
- **进度与标题**：设置标题、进度值/颜色、信息文本
- **控制回调**：设置停止回调（setOnStopCallback）

## 显示效果

> 悬浮窗日志模块，由计时器、任务标题区、缩放按钮、日志内容区、进度条、底部栏、关闭按钮等部分组成。每个部分均可自定义显示/隐藏，支持拖动、缩放、最小化/恢复等操作。

## API 接口列表

### 1. 窗口管理

#### setContext()

按需设置 Context（建议传入 applicationContext），在未调用 `UtilsMain.init` 时也可提前注入。

> setContext()和 UtilsMain.init() 二选一调用即可。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| context | Context | 是 | 应用或 Activity 的 context，建议 applicationContext |

| 返回值类型 | 说明 |
|-----------|------|
| void | 无 |

**示例：**
```lua
import('com.nx.assist.lua.LuaEngine')

local floatLogOps = UtilsMain.floatLog()

local context = LuaEngine.getContext()
floatLogOps.setContext(context)  -- 传入已有的 context
```

#### show()

在UI线程创建并异步显示悬浮窗口。

> 必须先设置上下文，才能显示。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| x | Int | 是 | 初始 X（px） |
| y | Int | 是 | 初始 Y（px） |
| w | Int | 是 | 初始宽（px） |
| h | Int | 是 | 初始高（px） |
| topTextSize | Float | 是 | 顶部计时器和标题文字大小 sp |
| bottomTextSize | Float | 是 | 底部信息栏文字大小 sp |
| isShowTimer | Boolean | 是 | 是否显示左上角计时器 |
| isShowTitle | Boolean | 是 | 是否显示任务信息标题 |
| isShowMinBtn | Boolean | 是 | 是否显示右上角最小化按钮 |
| isShowProgressBar | Boolean | 是 | 是否显示进度条 |
| isShowBottomBar | Boolean | 是 | 是否显示底部信息栏 |
| isShowControlBtn | Boolean | 是 | 是否显示右下角控制按钮 |
| isAllowDrag | Boolean | 是 | 是否允许标题栏拖动 |
| isSaveDragPosition | Boolean | 是 | 是否保存拖动位置(左边和顶边) |
| isSaveSize | Boolean | 是 | 是否保存窗口尺寸(宽度和高度) |

| 返回值类型 | 说明 |
|-----------|------|
| void | 无 |

**示例：**
```lua
local floatLogOps = UtilsMain.floatLog()
floatLogOps.show(100, 100, 800, 600, 12, 12, true, true, true, true, true, true, true, true, true)
```

| 参数名 | 类型 | 必填 | 说明 |

**示例：**

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| alpha | Int | 是 | 0-255 |

| 返回值类型 | 说明 |
|-----------|------|

```lua

```

#### setVisible()

完全显示或隐藏悬浮窗口（通过设置窗口 visibility 属性）。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| visible | Boolean | 是 | true=显示窗口，false=隐藏窗口 |

| 返回值类型 | 说明 |
|-----------|------|
| void | 无 |

**示例：**
```lua
local floatLogOps = UtilsMain.floatLog()
floatLogOps.setVisible(false)  -- 隐藏窗口（不占用屏幕空间）
sleep(3000)
floatLogOps.setVisible(true)   -- 显示窗口
```

#### toggleVisible()

切换悬浮窗口的完全显示/隐藏状态。

| 返回值类型 | 说明 |
|-----------|------|
| void | 无 |

**示例：**
```lua

floatLogOps.toggleVisible()  -- 如果窗口可见则隐藏，如果隐藏则显示
```

#### close()

关闭悬浮窗口并释放视图。

| 返回值类型 | 说明 |
|-----------|------|
| void | 无 |

**示例：**
```lua

```

| 返回值类型 | 说明 |
|-----------|------|
| String | 窗口状态 JSON |

**示例：**
```lua

```

### 2. 日志操作

#### addLog()

添加日志（文本、颜色、字号）。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| message | String? | 是 | 日志内容 |
| color | Int | 否 | 文本颜色 |
| textSize | Float | 否 | 字号 |
| backgroundColor | Int | 否 | 背景色 |
| padding | Int | 否 | 内边距(px) |

| 返回值类型 | 说明 |
|-----------|------|
| void | 无 |

**示例：**
```lua
local floatLogOps = UtilsMain.floatLog()
floatLogOps.addLog("xfxPlugin")
floatLogOps.addLog("hello", 0xFFFFFF, 12)
floatLogOps.addLog("warn", 0xFFFFAA00, 12, 0x22000000, 16)
```

#### clear()

清空日志内容。

| 返回值类型 | 说明 |
|-----------|------|
| void | 无 |

**示例：**
```lua

floatLogOps.clear()
```

```lua
import('com.nx.assist.lua.LuaEngine')
import('java.lang.Runnable')

local loader = LuaEngine.loadApk('xfxPlugin-release.apk')
if not loader then
    error('Failed to load APK')

local UtilsMain = loader.loadClass('com.xfx.plugin.UtilsMain')

local floatLogOps = UtilsMain.floatLog()

-- 设置上下文(或使用Main.init设置全局上下文)
local context = LuaEngine.getContext()

local screenWidth, screenHeight = getDisplaySize()
local widowHeight = math.floor(screenHeight / 4) -- 默认窗口高度

--创建绘制

floatLogOps.show(
    0,
    (screenHeight - widowHeight) / 2,
    screenWidth / 2,
    widowHeight,
    10,
    8,
    true,
    true,
    true,
    true,
    true,
    true,
    true,
    true,
    false
)
floatLogOps.setProgressBarColor(0xFFFF0000);

-- 关闭按钮回调
floatLogOps.setOnStopCallback(
    Runnable(
        {
            run = function()
                -- 在这里编写停止时要执行的代码
                print('停止执行')
                exitScript()
            end
        }
    )
)

floatLogOps.setInfo('关100，赞99，藏666')
for i = 1, 1000 do
    -- 判断是偶数，打印白色日志
    if i % 2 == 0 then
        floatLogOps.addLog('这是日志' .. i, 0xFFFFFFFF, 8) --#FFFF0000 取色工具 # 改 0x

        floatLogOps.addLog('这是日志' .. i, 0xFFFF0000, 9) --#FFFF0000 取色工具 # 改 0x
    end

    floatLogOps.setTitle('这是标题' .. i)
    -- // 更新进度
```

```lua
floatLogOps.show(100, 100, 600, 400, 12, 12, true, true, true, true, true, true, true, true, true)
floatLogOps.setTitle("测试完全隐藏功能")
floatLogOps.addLog("窗口已显示")
sleep(2000)

-- 方法1: 使用 setVisible 完全隐藏窗口（包括背景）
floatLogOps.addLog("2秒后完全隐藏窗口")
sleep(2000)
floatLogOps.setVisible(false)  -- 完全隐藏窗口
floatLogOps.addLog("这条日志你看不到，因为窗口已隐藏")
sleep(3000)

-- 重新显示窗口
floatLogOps.setVisible(true)
floatLogOps.addLog("窗口重新显示")
sleep(2000)

-- 方法2: 使用 toggleVisible 切换显示/隐藏
floatLogOps.addLog("使用 toggleVisible 切换")
floatLogOps.toggleVisible()  -- 隐藏
sleep(2000)
floatLogOps.toggleVisible()  -- 显示
floatLogOps.addLog("窗口再次显示")
sleep(2000)

-- 清理
floatLogOps.close()
```

**说明：**
- `setVisible(false)`: 完全隐藏窗口，包括背景和所有内容，不占用屏幕空间
- `setVisible(true)`: 重新显示之前隐藏的窗口
- `toggleVisible()`: 切换窗口的显示/隐藏状态


## 注意事项

- **初始化要求**：使用前需先调用 `xxxOps.setContext(context)`或`UtilsMain.init(context)` 初始化上下文。
- **线程约束**：所有方法在插件内部已强制在UI线程异步执行，无需在调用时，再次设置UI线程执行；
- **权限**：悬浮窗与 Toast 需系统悬浮窗权限（TYPE_APPLICATION_OVERLAY）；存储导出需读写权限。
- **性能**：日志量大时注意最大条数设置；
- **画面遮挡**：悬浮窗日志模块在屏幕上会遮挡下层的内容，并且不能穿透点击。在实际使用的时候，注意设置合适的位置和大小，避免遮挡重要内容导致无法识别（纯节点脚本无需考虑遮挡问题）。
