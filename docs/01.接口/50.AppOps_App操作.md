## 模块简介

`AppOps` 封装了应用层常用操作，包括安装状态判断、前后台判断、重启/启动/退出、版本与包信息获取、Root 权限判断、首次安装判断、安装/卸载应用等。

> ⚠️ 不支持懒人精灵`最高权限`下运行(最高权限下获取不到Context)。

## 功能概览

- **上下文设置**：设置 Context（setContext，与 UtilsMain.init 二选一）
- **UID 获取**：当前 App 或指定包名的 UID（getAppUid）
- **状态判断**：是否安装、Debug、系统应用、前台、首次安装、Root 权限（isAppInstalled/isAppDebug/isAppSystem/isAppForeground/isFirstTimeInstalled/isAppRoot）
- **启动/重启/退出**：重启自身（relaunchApp）、启动 App（launchApp）、打开设置（launchAppDetailsSettings）、退出（exitApp）
- **信息获取**：包名、名称、路径、版本号/码（getAppPackageName/getAppName/getAppPath/getAppVersionName/getAppVersionCode）；已安装应用间 appName 与 packageName 互查（getPackageName）；指定应用完整信息（getAppInfo）
- **安装/卸载**：系统安装器安装（installApp）、系统卸载器卸载（uninstallApp）；静默安装应用（installAppSilent）、静默卸载应用（uninstallAppSilent，需要 Root 权限或系统权限）

## API 接口列表


#### setContext()

按需设置 Context（建议传入 applicationContext），在未调用 `UtilsMain.init` 时也可提前注入。

> setContext()和 UtilsMain.init() 二选一调用即可。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| context | Context | 是 | 应用或 Activity 的 context，建议 applicationContext |

| 返回值类型 | 说明 |
|-----------|------|
| void | 无 |

**示例：**
```lua
import('com.nx.assist.lua.LuaEngine')

local softOps = UtilsMain.appOps()

local context = LuaEngine.getContext()
softOps.setContext(context)  -- 传入已有的 context
```

#### installAppSilent()

静默安装应用。无 Root 权限时必须持有 `android:sharedUserId="android.uid.shell"` 和 `INSTALL_PACKAGES` 权限。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| filePath | String? | 是 | APK 文件路径 |

| 返回值类型 | 说明 |
|-----------|------|
| Boolean | 成功返回 true，失败返回 false |

**示例：**
```lua
local softOps = UtilsMain.appOps()
-- 静默安装应用（需要 Root 权限或系统权限）
local success = softOps.installAppSilent("/sdcard/app.apk")
print("安装结果: " .. tostring(success))
```

#### installAppSilent()

静默安装应用（带参数）。依赖 Root 权限。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| filePath | String? | 是 | APK 文件路径 |
| params | String? | 否 | 安装参数（例如 "-r" 表示替换已安装应用，"-s" 表示安装到 SD 卡） |

| 返回值类型 | 说明 |
|-----------|------|
| Boolean | 成功返回 true，失败返回 false |

**示例：**
```lua
local softOps = UtilsMain.appOps()
-- 静默安装应用，替换已安装的应用（需要 Root 权限或系统权限）
local success = softOps.installAppSilent("/sdcard/app.apk", "-r")
print("安装结果: " .. tostring(success))
```

#### uninstallAppSilent()

静默卸载应用。依赖 Root 权限。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| packageName | String? | 是 | 目标包名 |

| 返回值类型 | 说明 |
|-----------|------|
| Boolean | 成功返回 true，失败返回 false |

**示例：**
```lua

```

#### uninstallAppSilent()

静默卸载应用（保留数据）。无 Root 权限时必须持有 `android:sharedUserId="android.uid.shell"` 和 `DELETE_PACKAGES` 权限。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| packageName | String? | 是 | 目标包名 |
| isKeepData | Boolean | 是 | 是否保留数据，true 保留，false 不保留 |

| 返回值类型 | 说明 |
|-----------|------|
| Boolean | 成功返回 true，失败返回 false |

**示例：**
```lua
local softOps = UtilsMain.appOps()
-- 静默卸载应用，保留数据（需要 Root 权限或系统权限）
local success = softOps.uninstallAppSilent("com.example.app", true)
print("卸载结果: " .. tostring(success))
```

|--------|------|------|------|
| filePath | String | 是 | APK 文件路径 |

| 返回值类型 | 说明 |
|-----------|------|

|--------|------|------|------|

| 返回值类型 | 说明 |
|-----------|------|
| String | 包名；未找到返回空字符串 |

**示例：**
```lua

```

#### getAppInfo(packageName: String)

获取指定包名的 App 完整信息（JSON 格式）。

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| packageName | String | 是 | 目标包名 |

| 返回值类型 | 说明 |
|-----------|------|
| String | JSON 字符串，包含 packageName、name、packagePath、versionName、versionCode、minSdkVersion、targetSdkVersion、isSystem 字段；失败返回空字符串 |

**JSON 字段说明：**

| 字段 | 类型 | 说明 |
|------|------|------|
| packageName | String | 包名 |
| name | String | 应用显示名称 |
| packagePath | String | APK 文件路径 |
| versionName | String | 版本名 |
| versionCode | Int | 版本码 |
| minSdkVersion | Int | 最低 SDK 版本（Android 7.0+，否则返回 -1） |
| targetSdkVersion | Int | 目标 SDK 版本 |
| isSystem | Boolean | 是否为系统应用 |

**示例：**
```lua
local softOps = UtilsMain.appOps()

-- 获取指定应用信息
local json = softOps.getAppInfo("com.tencent.mm")
if json ~= "" then
    local info = jsonLib.decode(json)
    print("包名: " .. info.packageName)
    print("名称: " .. info.name)
    print("路径: " .. info.packagePath)
    print("版本名: " .. info.versionName)
    print("版本码: " .. tostring(info.versionCode))
    print("最低 SDK: " .. tostring(info.minSdkVersion))
    print("目标 SDK: " .. tostring(info.targetSdkVersion))
    print("是否系统应用: " .. tostring(info.isSystem))
end
```

## 完整示例

```lua
import('com.nx.assist.lua.LuaEngine')

-- 加载 APK 文件
local loader = LuaEngine.loadApk('xfxPlugin-release.apk')
if not loader then error('Failed to load APK') end

-- 获取上下文
local context = LuaEngine.getContext()

-- 加载插件主类
local UtilsMain = loader.loadClass('com.xfx.plugin.UtilsMain')
if not UtilsMain then error('Failed to load Class') end

-- -- 初始化插件
-- UtilsMain.init(context)

-- 获取 App 工具
local softOps = UtilsMain.appOps()
softOps.setContext(context)  -- 传入已有的 context

print("=== 基本信息 ===")
print("包名: " .. softOps.getAppPackageName())
print("名称: " .. softOps.getAppName())
print("路径: " .. softOps.getAppPath())
print("版本名: " .. softOps.getAppVersionName())
print("版本码: " .. tostring(softOps.getAppVersionCode()))
print("UID: " .. tostring(softOps.getAppUid()))
print("前台?: " .. tostring(softOps.isAppForeground()))

print("\\n=== 启动与设置 ===")
local targetPkg = "com.example.app"
if softOps.isAppInstalled(targetPkg) then
    softOps.launchApp(targetPkg)
else
    -- 打开自身的应用信息页，提示用户安装目标应用
    softOps.launchAppDetailsSettings(softOps.getAppPackageName())
end

print("\\n=== 重启与退出（谨慎调用） ===")
-- softOps.relaunchApp(false)  -- 如需测试可取消注释
-- softOps.exitApp()           -- 如需测试可取消注释
```


## OOP 调用例子

```lua
-- ============================================
-- App 操作示例
-- 演示如何使用 XfxPlugin.lua 调用 softOps 进行应用管理
-- ============================================

print('========== App 操作示例 ==========')

import('com.nx.assist.lua.LuaEngine')

-- 加载 XfxPlugin.lua 类
local xfxModule = require('lib/XfxPlugin')

-- 创建 XFX 对象实例
local XFX = xfxModule:new({
    apkName = 'xfxPlugin-release.apk',  -- APK 文件名
    -- context = LuaEngine.getContext(),  -- 上下文对象（不支持最高权限）
})

-- 获取 softOps 对象
local softOps = XFX:getOps('appOps')
if not softOps then
    print('无法获取 softOps 对象')
    return
end

softOps.setContext(LuaEngine.getContext())  -- 设置上下文对象

print('开始 App 操作示例...')

-- ============================================
-- 1. 获取当前 App 信息
-- ============================================
print('\n--- 1. 获取当前 App 信息 ---')

local packageName = softOps.getAppPackageName()
local appName = softOps.getAppName()
local appPath = softOps.getAppPath()
local versionName = softOps.getAppVersionName()
local versionCode = softOps.getAppVersionCode()
local uid = softOps.getAppUid()

print('包名: ' .. packageName)
print('名称: ' .. appName)
print('路径: ' .. appPath)
print('版本名: ' .. versionName)
print('版本码: ' .. tostring(versionCode))
print('UID: ' .. tostring(uid))

-- ============================================
-- 2. 判断当前 App 状态
-- ============================================
print('\n--- 2. 判断当前 App 状态 ---')

local isDebug = softOps.isAppDebug()
local isSystem = softOps.isAppSystem()
local isForeground = softOps.isAppForeground()

print('是否 Debug 版本: ' .. tostring(isDebug))
print('是否系统应用: ' .. tostring(isSystem))
print('是否在前台: ' .. tostring(isForeground))

-- ============================================
-- 3. 检查其他 App 是否安装
-- ============================================
print('\n--- 3. 检查其他 App 是否安装 ---')

-- 检查微信是否安装
local wechatPkg = 'com.tencent.mm'
local wechatInstalled = softOps.isAppInstalled(wechatPkg)
print('微信是否安装: ' .. tostring(wechatInstalled))

-- 检查 QQ 是否安装
local qqPkg = 'com.tencent.mobileqq'
local qqInstalled = softOps.isAppInstalled(qqPkg)
print('QQ 是否安装: ' .. tostring(qqInstalled))

-- 检查不存在的应用
local fakePkg = 'com.example.fake'
local fakeInstalled = softOps.isAppInstalled(fakePkg)
print('不存在的应用是否安装: ' .. tostring(fakeInstalled))


-- ============================================
-- 5. 获取其他 App 的 UID
-- ============================================
print('\n--- 5. 获取其他 App 的 UID ---')

if wechatInstalled then
    local wechatUid = softOps.getAppUid(wechatPkg)
    print('微信 UID: ' .. tostring(wechatUid))
end

-- ============================================
-- 6. 启动其他 App
-- ============================================
print('\n--- 6. 启动其他 App ---')

-- 注意：启动其他应用需要相应权限，请谨慎使用
if wechatInstalled then
    print('准备启动微信...')
    softOps.launchApp(wechatPkg)
    sleep(5000)
    
    -- -- 检查是否成功启动
    -- local running = softOps.isAppRunning(wechatPkg)
    -- print('微信是否运行: ' .. tostring(running))
else
    print('微信未安装，无法启动')
end

-- ============================================
-- 7. 打开 App 设置页面
-- ============================================
print('\n--- 7. 打开 App 设置页面 ---')

if wechatInstalled then
    print('准备打开微信设置页面...')
    softOps.launchAppDetailsSettings(wechatPkg)
    print('已打开微信设置页面')
    sleep(5000)
end

-- ============================================
-- 8. 重新启动当前 App
-- ============================================
print('\n--- 8. 重新启动当前 App ---')

-- 注意：此操作会重启当前应用，请谨慎使用
print('注意：重新启动应用会终止当前脚本执行')
```

## 注意事项

- **初始化要求**：使用前需先调用 `xxxOps.setContext(context)`或`UtilsMain.init(context)` 初始化上下文。
- **权限**：
  - 启动其他应用或打开设置页通常不需要额外权限
  - `installApp()` 在 Target API > 25 时需要 `REQUEST_INSTALL_PACKAGES` 权限
  - `uninstallApp()` 在 Target API > 25 时需要 `REQUEST_DELETE_PACKAGES` 权限
  - `installAppSilent()` 和 `uninstallAppSilent()` 需要 Root 权限或系统权限：
    - 无 Root 权限时必须持有 `android:sharedUserId="android.uid.shell"` 和相应的安装/卸载权限
    - 有 Root 权限时可直接使用
- **静默安装/卸载**：这些操作会直接执行，不会弹出用户确认对话框，请谨慎使用。
- **普通安装/卸载**：`installApp()` 和 `uninstallApp()` 会弹出系统安装/卸载界面，需用户手动确认。
